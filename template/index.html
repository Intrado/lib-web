The window class

Each main section of the content can be given the class of 'window' (like this one has) this will help be able to group style each section of the layout. They can be styled to suit theme, these basic ones just have a border a slight bottom margin and some padding. These sections can be filled with elements of your choice, like forms, lists and suchlike.
