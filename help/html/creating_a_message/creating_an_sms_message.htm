Creating an SMS Message

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

SMS messages are text message that can be sent to cell phones. They may be created and stored using the Message Builder, shown below. You may also send an SMS message directly from the Job Editor. Just click on the Notifications tab followed by the Messages subtab to get to the Message Builder. Then click the Create SMS Message button the My SMS Messages section of the page.

SMS Messages must not exceed 160 characters in length. The counter at the bottom of the SMS Message Editor, shown below, can help you ensure that your message fits within the size restraints. When you have finished entering your message, click the Save button to return to the Message Builder.
