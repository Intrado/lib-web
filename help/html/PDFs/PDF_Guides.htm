PDF Guides

We've created several printable PDF guides to help you learn how to use SchoolMessenger or teach others.

Important: Some of these guides contain instructions for features you may not have access to or which may not currently be part of your account. For access to additional features, please contact your system administrator.  

Introduction

  • Quick Start for New Users: A simple, introductory walk-through guide where new users learn to configure their user account and then set up and send a test voice notification.

User Training Guides

  • Getting Started: This PDF has everything new users need to know about setting up and sending a broadcast; including logging in, creating a list, creating messages, and scheduling.

  • Advanced Training: This guide contains some of the more advanced features you can use to enhance your notifications.

  • Creating a Survey: This worksheet and guide helps you plan and prepare a survey.

System Administrator Training Guide

  • System Administrator Training: This PDF guide is intended for system administrators and covers topics such as user management, system configuration, and data imports.

  • Phone Opt-Out Guide: Instructions and advice for creating an intro message and campaign to inform phone message recipients of how to request to be opted-out of future calls. Includes instructions for setting up the intro message, running opt-out reports and putting numbers on your accounts block list.

  • SMS Text Opt-In Campaign: A very informative guide to help your organization effectively and efficiently prepare for and launch the "SMS Text Messaging Opt-In" campaign required to begin sending text messages through SchoolMessenger.

Pocket Guides

  • Remote Access Pocket Guide: This PDF contains pocket sized references for logging in and using the system by phone.
