Posting to an RSS Feed

The RSS messaging component allows you to syndicate your messages in feeds which people may subscribe to. Publishing messages to an RSS feed is easy. You can create feed content while creating your other message components in MessageSender or while creating reusable messages in Message Builder.

Creating RSS Feed Content

Including RSS Feed content  in your Broadcast is as simple as including any other destination type.

From the Dashboard

  1. Click the New Broadcast button.

  2. At the Message Content step, select the Add Social option and select  Post to RSS.

  3. Enter your Post Title and message.

  4. Select the category best describing your post

  5.  Click Save Social Messages and continue adding content for the other destinations you wish to include.

Advanced Posting

If you would like to create your social media posts using advanced features such as the Page message type with embedded Page Media, you should create a message from Message Builder and use the features of the Message Editor to do so. If you've never created an Advanced Message, you should start by reading the Create a Message help topic. For details on adding RSS Feed content to an Advanced Message, you should read the RSS Feed Content help topic.
