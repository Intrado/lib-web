Remote Telephone Access

If your system administrator has enabled this feature on your account, you may call in to your SchoolMessenger ASP Plus account to record your message and create your job. This easy-to-use feature supports simple messages (an unchanging message with a basic phone recording) and allows you to record and create multilingual jobs. In order to use this feature you must already have the list you want to call saved in the system.

When you call in to the system to create your job, you will be guided through the job creation process with a series of prompts. It is a good idea to be prepared by having the following items:

  1. The name of the list currently in the system that you want to call.

  2. Your message in whichever languages it needs to be recorded in.

  3. The priority level you want to set for your job (your account's default or highest priority).

  4. The number of days you would like your job to run.
