Remote Telephone Access

If your system administrator has enabled this feature on your account, you may call in to your system to record your message and create your Broadcast. This easy-to-use feature supports simple messages (an unchanging message with a basic phone recording) and allows you to record and create multilingual Broadcasts. In order to use this feature you must already have the list you want to call saved in the system.

When you call in to the system to create your Broadcast, you will be guided through the Broadcast creation process with a series of prompts. It is a good idea to be prepared by having the following items:

  1. The name of the list currently in the system that you want to call.

  2. Your message in whichever languages it needs to be recorded in.

  3. The priority level you want to set for your Broadcast (your account's default or highest priority).

  4. The number of days you would like your Broadcast to run.
