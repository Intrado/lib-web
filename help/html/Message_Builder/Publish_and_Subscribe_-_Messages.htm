Publish and Subscribe - Messages

Important: Some menus or features shown may not appear in your account, as only those to which you have been granted access will be visible.  For access to additional features, please contact your account administrator.

The Publish and Subscribe feature can help you save time by sharing lists and messages that may be useful to other users of your account. This feature is intended to save you time by allowing complicated, manually entered lists or messages to be shared with other users who need them.

Who should be a publisher?

If you have a user who has manually created a list of emergency contacts for your district and multiple users need to be able to use this list, enable publishing on this user's account. Not all users should be publishers since most lists are simple rule-based lists which are easily created.

Publishing

If you have a list or message that you would like to allow other users to use, you may publish the list by clicking on the Publish link next to your list message on the List Builder or Message Builder screen. You will be taken through a simple step-by-step process to publish the list. Click the Guide button within the process for a thorough explanation of each step.

Note: Once your list is published, any changes you make to that list or message can affect Broadcasts created by other users using your list.

Subscribing

If you would like to subscribe to a list or message, simply go to the List Builder or Message Builder and click the Subscribe button. You will taken to a screen containing a list of lists or messages you may subscribe to. Simply click the Subscribe link next to the item you want.

Note: When using a list or message you've subscribed to, be aware that if a change occurs to the list or message, it can affect your Broadcasts which contain the list or message.
