You will need to have an authorized Facebook account to post to Facebook. To learn more about managing your Facebook account, please view the Managing Pages help topic.

Facebook Message Editor

To use this tutorial, you should first follow the steps in Create a Message. If you have done this, you will be in the Message Editor where you can add new content using the Message Content grid.

  1. Click the green plus icon next to "Facebook" in the Message Content grid.

  2. Enter your message in the "Facebook Message" box.

  3. Click "Spell Check" if desired.

  4. Click “Done”.
