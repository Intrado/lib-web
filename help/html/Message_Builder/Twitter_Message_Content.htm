Before Posting to Twitter

You will need to have an authorized Twitter account to post to Twitter. To learn more about managing your Twitter account, please view the Twitter help topic.

Twitter Message Editor

To use this tutorial, you should first follow the steps in Create a Message. If you have done this, you will be in the Message Editor where you can add new content using the Message Content grid.

  1. Click the green plus icon next to Twitter in the Message Content grid,

  2. Enter your message in the Twitter Message Editor and click “Done”.

Note: A short link to longer Page message content will be automatically attached to your Twitter post when you create a Page. If you have not created a Page, space for the link is still reserved. For this reason, Twitter posts are slightly shorter than the typical 140 character limit.
