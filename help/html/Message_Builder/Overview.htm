About Message Builder

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

What is Message Builder?

Message Builder helps you manage messages in the system. From here you can:

  • View, edit, or delete messages.

  • Publish and subscribe to messages.

  • Create new messages using all of the advanced messaging capabilities of the system.

With Message Builder, you can easily create messages containing content for any combination of destinations as well as access advanced features for common destinations, such as the ability to personalize your phone and email content with data.  You can also create Pages with downloadable content to attach to your social media posts and RSS feeds.

Accessing Message Builder

You can access Message Builder by clicking on the Broadcasts tab followed by the Messages subtab.

View, Edit or Delete Messages

View

To view the content of a message, click the message name in My Messages  to go to the Message Editor. Alternatively, if you want to preview a specific content type, click on the icon for the content type next to the message creation time in My Messages.

Edit

Clicking the Edit action link to the right of a message in My Messages will also take you to the Message Editor.

Delete

Click the Delete action link will delete the message from your message list.

Publishing or Subscribing to Messages

The Publish and Subscribe feature lets you share lists and messages with other users in the system. To learn more about, please view the Publish and Subscribe Overview.
