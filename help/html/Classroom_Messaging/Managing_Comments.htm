Managing Comments

Important: This feature is only visible if your system includes the Classroom Messaging feature. For more information regarding the Classroom Messaging feature, please call 1-800-920-3897.

While there are many premade comments in Classroom Messaging, you may find that some don't fit your school or that you need a custom comment that isn't included. You may also want to organize comments so that it's easier to find the ones you use frequently.

The Classroom Message Manager

All organizing and editing of Classroom Comments is performed through the Classroom Message Manager, shown above.

Access the Classroom Message Manager:

  1. Click on the Admin tab followed by the Settings subtab.

  2. Click the Classroom Message Manager link in the Features section.

 

Organizing Comments

This section will cover creating categories for comments, moving comments into the new categories, and disabling/enabling comments.

From within the Classroom Message Manager:

Create a category for comments

  1. Click the Create Category button.

  2. Enter a name and select an icon to associate with your new category.

  3. Click the Save button when you're done.

  4. Repeat for any new categories you need.

Move comments into the new categories

  1. Find the comment you want to move.

  2. Select the new category from the Move To dropdown menu to the right of the comment.

Disabling/Enabling comments

If you find a comment you don't want to have available, uncheck the checkbox to the left of the comment. To enable the comment, just check the box again.
