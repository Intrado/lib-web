Quickly Send a Basic Message

This topic assumes:

  • You are completely unfamiliar with the system and need to quickly send out a message.

  • Your account does not have any Broadcast Templates configured for the Broadcast you need to send. (This is just because Broadcast Templates let you quickly send a Broadcast which has already been configured.)

  • You need to send a simple message to a large group, such as an entire school or grade, which can be easily selected without having a premade list. If you need to send a message which requires a more complicated list or a message using advanced features like dynamic data inserts, please refer to the Getting Started and/or Advanced Training PDF guides.

 

From the Dashboard

When you log into the system, the first screen you see is called the Dashboard. On the right side, there is a large button labeled "New Broadcast", shown below. Click this button to get started.

 

Subject and Recipients

The following screenshot shows you the first step for creating a new Broadcast.

Broadcast Info

  • Enter a Subject. This will be the name of your Broadcast and how you will see it in your Report.

  • Select a Type. This is the type of message you are sending, such as "General Announcement" or "Emergency". This affects where recipients are contacted so be sure to choose the appropriate type.

Note: The Type you choose is important because it will affect how message recipients are contacted. For example, some recipients only want to receive actual emergency messages at a certain phone number which is associated with Emergency jobs.

Recipient List

  • Click "Add Message Recipients".

  • Select "Custom Rules", as shown above, and begin creating rules to select the group of recipients you need. Click Save after adding each rule.

Simple List Examples

    • Calling an entire school. Select the "School" field, then check the box next to the school name in the list of values. Click Save followed by Done to use this list.

    • Calling all 10th graders in one school. Select the "School" field, then check the box next to the school name in the list of values. Click Save then click Add Rule. Select the "Grade" field then check the value "10". Click Save followed by Done to use this list.

  • Click "Add Myself" and enter your contact information if you would like to be contacted when you send your Broadcast.

  • If you want to save the list you've created so you can reuse it later, click the folder icon to the left of the list. You can preview your list by clicking the magnifying glass icon. Click the "X" if you want to delete the list and start over.

  • Click the "Continue" button when you're done creating a list.

Message Content

In this step you will create your message content for all mediums you would like to include in your Broadcast. The following screenshot shows Phone message content being added. The message entered in the Scratch Pad area will act as a script for recording the message. You can use this feature if you find it helpful to prepare. Anything typed in the Scratch Pad won't be saved.

Add Phone Content

Note: This section assumes you want to record your voice. If you'd prefer to use text-to-speech, click the Text-to-Speech button in the Voice Type options. This option lets you type your message content to be read by the system. You may also use the automatic translation feature with Text-to-Speech.

  • You can optionally create a script for yourself using the Scratch Pad before recording your message. This isn't saved anywhere. It's just to help you organize and prepare.

  • Enter the phone number where you may be reached to record your message in the Recording field.

  • Click "Call Me to Record". The system will call you at the number you've entered. Simply follow the prompts to record your message.

  • After you've finished the phone message, you may click "Add Email" or any other destination type available to your account and add content. If you just want to send your phone message, skip this step.

  • Click the "Continue" button when you are done creating your message content.

Step 3: Review and Send

  • Click "Send Now" to send your message immediately if everything looks correct.

The system will begin sending your message to the recipients on your list. You can monitor the progress from the Dashboard after clicking "Send Now". You'll see your job running in the "In Progress" section. It will move to "Completed" once it's done. View the help topics on creating reports if you would like to generate a report for this job.
