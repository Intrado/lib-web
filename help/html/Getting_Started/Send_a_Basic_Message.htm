Send a Basic Message

This topic assumes:

  • You are completely unfamiliar with the system and need to quickly send out a message.
  • You need to send a simple message to a large group, such as an entire school or grade, which can easily be selected without having a premade list.

Note: If you need to make a specialized list, step by step instructions can be found in the Getting_Started PDF guide or by navigating to Notifications>Lists, clicking "Create a New List", and then clicking on the Guide help button for inline help. If you need to create a special list, you need to do so before continuing.

If you need to send a message with complicated components like dynamic data inserts, create lists with the advanced list building tools, or do anything that is not covered here, you should review the Getting Started and/or Advanced Training PDF guides.

Using MessageSender

On the Start Page when you log in, you will see a large blue button that says "Easy Start" and an orange button labeled "Emergency". If the message you are trying to send is really an emergency, use the Emergency button, otherwise click on Easy Start.

Note: This choice is important because it will affect how message recipients are contacted. For example, some recipients only want to receive actual emergency messages at a certain phone number which is associated with Emergency jobs.

  • Click the appropriate button for your message to continue to the next step.

Step 1: Getting Started

  • Click on the "Guide" button on the right side of the screen for inline help that will follow you through MessageSender. You may also get more information by mousing over the label associated with any field. This help topic will continue to describe the rest of the process, but you may also find that the Guide help is enough to complete the message sending process. Note: The Guide is available throughout the system so you're never far from help.

  • Enter a Job Name. You will see this on your report.

  • Select a Type. This is the type of message you are sending, such as "General Announcement" or "Emergency". This affects where recipients are contacted so choose the appropriate type.

  • Select a Creation Method. If you are just sending a simple phone message, select Record. The email component will be automatically generated for you. This is the easiest way to send a message.

  • Click the "Next" button at the bottom of the screen when you are done.

Step 2: List

  • Click "Build List Using Rules". Add rules that describe who you need to contact. For example, if you need to contact an entire school, select the "School" field and the school's name from the value column, then click "Add" to confirm the rule. If you need to select a smaller subset of the school, such as only 6th graders, select "Grade" is "6", followed by the "Add" button. When you are done adding rules, click the "Done Adding Rules" button.

  • You may wish to add yourself so the system will contact you too. If so, simply click "Add Myself" and enter your contact information.

  • Click the "Next" button when you're done creating a list.

Step 3: Record

  • Enter the phone number where you may be reached to record your message.

  • Click "Call Me to Record". The system will call you at this number. Simply follow the prompts to record your message.

  • If you want to record in another language, you can select the language after recording the English message, then have the system call you again.

  • Click the "Next" button when you are done.

Step 4: Social Media

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

  • If this feature is enabled on your account, you can select social media sites where your message will appear. Select which sites, if any, and click "Next".

Step 5: Job Options

  • Click "Now" to send your message immediately.

Step 6: Review and Confirm

  • Check the "Confirm" checkbox.

  • Click the "Next button.

The system will begin sending your message to the recipients on your list. You can monitor the progress from the Start Page after clicking "Done". You'll see your job on the Jobs Timeline and in Recent Activity. View the help topics on creating reports if you would like to generate a report for this job.
