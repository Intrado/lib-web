Contact Manager Settings

Important: This feature is only visible if your system includes the Contact Manager feature. The Contact Manager is a new feature from SchoolMessenger that allows your recipients to control the ways in which they may be contacted. For more information regarding Contact Manager, please call 1-800-527-5225.

If enabled on your account, you may manage the systemwide settings for Contact Manager from the Settings subtab.

Editing Contact Manager's Settings:

  1. Click the Admin tab followed by the Settings subtab.

  2. Click the Contact Manager Settings link on the right side of the page.

 

Contact Manager Setting Options:

  • Activation Code Lifetime - Enter the number of days that an activation code should be valid for.

  • Allow activation via phone - Check this box to allow contact manager users to activate their accounts by phone. Activation codes are not needed for activation by phone.

  • Require phone numbers for Emergency and High Priority Job Types - Check this box if you would like to ensure that every recipient has at lease one phone number available for these job types.

  • Restricted Destination Fields - Select any fields which Contact Manager users should not be allowed to edit.
