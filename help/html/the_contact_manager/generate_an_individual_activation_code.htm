Generate an Individual Activation Code

Important: This feature is only visible if your system includes the Contact Manager feature. The Contact Manager is a new feature from SchoolMessenger that allows your recipients to control the ways in which they may be contacted. For more information regarding Contact Manager, please call 1-800-527-5225.

In addition to generating bulk activation codes for groups of Contact Manager users, you may also generate single codes for individuals. This feature is good for enabling new students or staff members to use the Contact Manager.

 

How to Generate an Individual Activation Code

  1. Click on the System Tab followed by the Contacts subtab.

  2. Click on the Manage Activation Codes button.

 

  1. Click the By Person radio button in the Search section of the Activation Code Manager as shown above. If you don't have this information, you may also search by their name using By Criteria.

  2. Enter the required information in the fields - person ID if you are searching By Person or a name if you are using criteria.

  3. Click the magnifying glass icon next to the person's name in the Search Results to go to the person's Contact Information page.

  4. Click the Generate Activation Code button at the bottom of the Contact Information, shown below.

  5. Make sure to notify the individual as to what their Activation Code is.
