About Secure Document Delivery (SDD)

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

What is SDD?

Many organizations use tools which produce large PDF reports filled with individual reports for each member that includes the member's unique ID number. For example, a school's student information system may output one large PDF file with report cards for each student. Each report card will have the student's ID number somewhere on the page. These sorts of reports are typically printed and mailed to each individual.  

Secure Document Delivery allows you to automatically slice (burst) these large PDF files apart for delivery by email. SDD matches each individual to their portion of the PDF report using their ID number. Additionally, SDD can protect each individual portion so that the recipient must enter a password to open the file.  

What's a "Document"?

A Document is created by pairing your PDF report with a Template. Templates are sets of rules used by SDD to burst your PDF file into individual reports for delivery. Once a template is created for a type of PDF file, it can be reused for future reports so long as the layout of the report has not changed.

How do I get a Template for my PDF file?

Templates are generated by our support representatives. To facilitate template creation, you will need to be able to provide at least a portion of the PDF file you wish to burst. Once you have generated this PDF file, contact support and  a representative will coordinate with you to create a template within 48 hours.

Note: If you would like to use SDD to deliver  report cards, feel free to contact support to create your templates well in advance of the end of the quarter or semester. Remember, once your template has been created, you may use it for report cards repeatedly. The template will be valid until your PDF layout changes.
