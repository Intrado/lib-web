Emergency Preparedness

Are you ready for an emergency? This Emergency Preparedness Guide can help you make sure that you are.

Before an Emergency Happens:

  • Create a List of all of the students in your school.

  • Know your Remote Access phone number, User ID, and PIN.

  • Have a copy of the Remote Access Pocket Guide in an easily accessible location.

  • Practice using the system by using it to call yourself. Become familiar with how to use EasyCall and Remote Access.

 

During an Emergency

  • Get your Remote Access Pocket Guide from its easily accessible location to help you through the Remote Access steps.

  • Call your system's Remote Access phone number and follow the prompts to send your message to the list of students you prepared before the emergency.

  • Make sure you set your job to the highest priority so that the system knows to pause any lower priority jobs to get your message out immediately.
