The Contact Database

The Contact Database page, found under the System tab, allows you to see all of the contacts in the system's database and edit their contact details. This feature is safe to turn on for any user account because the data that is visible is restricted by the user's Data View. For example, a user that is restricted to only seeing one school's data would only see the students from that school on this page while a system administrator would be able to see everyone in a district. The Contact Database page is a great way to check the contact information for a particular student, estimate how large a certain list would be, or even see how many students are in the system for a particular school.

Using Rules to Search the Contact Database

Contacts can be displayed in their entirety or searched by using rules. Rules can be used to find groups of students meeting certain criteria or specific individuals.

To search with rules:

  1. Select the field that you wish to filter by.

  2. Select the data items that you wish to include in your rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking.

  3. Click the Add button and repeat steps 1 through 3 until all necessary rules are added.

If you want to find a specific person in the database, try using "Last Name" "starts with" and then typing his/her name in the third field. You may then include "First Name" "starts with" with his/her name in the third field to filter the results to only that name combination.
