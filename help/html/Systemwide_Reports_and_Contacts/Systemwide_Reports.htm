Systemwide Reports

SchoolMessenger contains systemwide reporting features that help you know exactly what your system is doing at any given time including:

  • Active & Pending Jobs

  • Completed Jobs

  • Repeating Jobs

 

Active & Pending Jobs

The systemwide Active & Pending Jobs screen shown below can be accessed by clicking the System tab followed by the Active Jobs tab. This screen shows all of the jobs that are scheduled or currently running for every user. It contains several columns of information.

  • Submitted by - This column shows which user submitted which jobs.

  • Job Name - This column shows the name of the job that is active or pending.

  • Status - This column shows whether or not the job has been submitted as well as if it is active or cancelled.

  • Total - This shows the number of people who will be contacted by this job and what types of communication will be used (phone or email).

  • Remaining - This column shows how many people have not yet been contacted by this job.

  • Scheduled Start - This column shows when the job is scheduled to run.

  • Actions - This column contains two links. Clicking Login as this User will log you in as that use so that you may edit the job and Cancel will cancel the job so that it stops running or does not run at all.

 

Completed Jobs

The systemwide Completed Jobs screen shown below displays all of the jobs that the system has completed for every user. It can be accessed by clicking on the System tab followed by the Completed Jobs tab. It contains several columns of information explained below.

  • Submitted by - This column shows which user submitted which jobs.

  • Job Name - This column shows the name of the job that was completed.

  • Start Date - This column shows the date when the job ran.

  • Status - This column shows if the job was complete,

  • Total - This column shows the number of people contacted and by what method.

  • Success Rate - The Success Rate shows how many messages were successfully delivered (ie. phone calls were answered by a person or machine versus no answer at all).

  • Scheduled End Date/Time - This column shows when the job was scheduled to stop running whether or not it was finished.

  • Actions - This column contains three links. Login as this User lets you log into the user account which created the job, report lets you see more details about the job, and archive which allows you to archive the job, causing it to no longer display on the user's Completed Jobs page.

 

Repeating Jobs

The systemwide Repeating Jobs screen shown below displays all of the repeating jobs currently scheduled in the system. It can be accessed by clicking the System tab followed by the Repeating Jobs tab. It contains several columns of information explained below.

  • Submitted by - This column shows which user submitted which jobs.

  • Job Name - This column shows the name of the job that was completed.

  • Next Scheduled Run - This column shows the next time that the job is scheduled to run.

  • Actions - The column contains two links. The first, Login as this user (or Edit if it is your own job) allows you to make changes to the job. The second, Delete, removes the job from the system.
