Notification Monitoring

Notification monitoring allows you to configure monitors which will notify you when certain kinds of jobs are sent from the system. You may monitor by job event, such as whenever a job is submitted, or job type, such as "Emergency". The monitor can be restricted to monitoring certain users or alert you to the notifications of all users.

Configuring a Monitor

Navigate to System>Monitors and click the "Add Monitor" button.

  1. Select a job event from the Job Event menu. If you select a Job Event, but no additional filters, you will be notified of all events of that type in the system.

  2. If you would like to restrict the monitor to only certain job types, select a job type from the Job Types options.

  3. To restrict the monitor to specific users, select one or more users from the list of Users.

You will now receive an email every time this job event occurs. If you wish to change or delete a monitor, simply return to System>Monitors and select "Edit" or "Delete" for the monitor.
