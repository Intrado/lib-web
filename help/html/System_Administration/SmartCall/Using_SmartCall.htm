Using SmartCall

This feature allows you to use your own phone system for day to day calling, but take advantage of the high-volume ASP phone system in the event of an emergency. Additionally, your database will be hosted on the ASP, thereby eliminating the need for an additional server and simplifying maintenance.

Note: This feature is for customers with SmartCall only.

Configuring SmartCall

  1. Click the Admin tab followed by the Settings subtab.

  2. Click the SmartCall link in the System section. You will be taken to the SmartCall Manager, shown below.

Status

The Status link will take you to a page, shown below, where you can see the current state of the SmartCall appliance.

Resource Schedule

Clicking the Resource Schedule Link in the SmartCall Manager will take you to a screen where you can limit the amount of local calling resources SmartCall can use. This allows other telephones to share the same resource as SmartCall.

Route Plan

This section is used for configuring custom routes. Some systems my require additional digits for certain prefixes. It can also be used to transform phone numbers or block numbers completely. The Route Plan Manager contains the following features:

At the top of the Route Plan Manager you will find an Upload Routes button which will allow you to upload a CSV file of your custom routes. The file should be formatted like the following:

[Match],[Strip],[Prefix],[Suffix (Optional)]

The Delete All button will clear all custom routes.

Important: Deleting is permanent.

Default Route Plan

This section contains the default route plan for all numbers which do not match a custom route.

Custom Route Plans

    • Match- This is the number that is being routed. It can be a whole phone number including the area code, just an area code, or an area code with prefix.

    • Strip- This field represents the number of additional characters the system should remove when it encounters the number in the Match field. The system can strip up to 99 characters.

    • Prefix - If certain digits need to be added to the beginning of certain phone numbers, they should be entered here.

    • Suffix - If certain digits need to be added to the end of certain phone numbers, they should be entered here.

 

Reset

Once you have configured your custom routes, click the reset button to restart SmartCall. Your new settings will take effect after the appliance has restarted.
