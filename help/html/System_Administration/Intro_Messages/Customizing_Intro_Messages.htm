Customizing Intro Messages

Intro Messages are played before your message when the recipient answers the phone. They include important instructions for pausing and replaying your message which contributes to greater success in communication.

You may wish to have a different intro message for emergency messages than regular messages. This can help alert recipients that the message regards an emergency.

Intro Message Best Practices

  • State your organization so the recipient knows who the call is from.

  • Instruct recipients that they must press 1 to hear the message and pound to place the call on hold.

  • Be brief.

Example Intro Messages

  • Default: "This is an important message from Springfield Independent School District. To hear this message now press 1. To place this call on hold press the pound key."

  • Emergency: "This is an emergency message from Springfield Independent School District. To hear this message now press 1. To place this call on hold press the pound key."

Changing Intro Messages

Intro messages are recorded just like a regular message, however since many systems have multilingual requirements, you may find it easiest to allow different users to record intros for each language. Any user whose Access Profile allows them to create new messages may record an intro message.

To change the intro message, for each language you wish to customize, simply select the user account which contains the message and the new intro message.

Once your users have recorded messages, you may select the user and message from the dropdown menus on the Intro Message Manager.
