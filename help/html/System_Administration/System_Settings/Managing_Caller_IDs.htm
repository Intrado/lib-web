Managing Caller IDs

The Caller ID numbers which users may associate with their respective accounts may be restricted to a specific list of approved numbers.

Note: If the user's Access Profile allows him to enter a custom Caller ID, he will be able to see the list of approved Caller ID numbers, but choose between using one of the numbers or entering a custom value.

Creating approved Caller ID numbers

  1. Navigate to Admin>Settings and click on Job Settings.

  2. If you would like to restrict users to the approved Caller ID numbers, check the "Approved Caller IDs" checkbox.

  3. Click the "Edit Approved Caller IDs" button to enter the Caller ID management screen.

  4. You may manually add one or more Caller ID numbers in the Manual Add field. Multiple Caller ID numbers should each be placed on a new line.

  5. If your users have other Caller ID numbers associated with their account or with certain jobs, these numbers will be displayed in the Import field. You may wish to add these numbers to your approved Caller IDs list. Just check the box next to any numbers you want to include.

  6. Click the "Add" button when you are finished adding and selecting Caller ID numbers. You will see the numbers in the Approved Caller ID area to the right.

  7. Click the "Done" button to finish and leave the Caller ID management screen.

Deleting approved Caller Id numbers

Simply navigate to the Caller ID management screen and click the "Delete" link next to the number in the Approved Caller ID are which you would like to remove.
