Blocking Phone Numbers

Important: Blocking a phone number or email address will result in this destination not being contacted for any reason by any user. It is a systemwide block.

Blocking certain phone numbers and email addresses from being contacted can easily be done by navigating to the Blocked Lists screen, found by clicking the System tab followed by the Blocked List subtab.

Viewing the Blocked Lists

After clicking the Blocked Lists subtab, click the Phone Calls / Text Messages Link, shown below to view the blocked list for phone numbers.

On Blocked List page you can view a list of all of the phone numbers which have been blocked by any user and the reason given for blocking the destination.

How to Block a Phone Number

  1. Navigate to the appropriate blocked list for phone numbers.

  2. Enter the destination information in the field provided. For phone numbers, you must include the area code.

  3. Enter a brief reason for blocking the number.

  4. Click the Add button.

How to Unblock a Phone Number

  1. Navigate to the appropriate blocked list for phone numbers.

  2. Find the destination you wish to unblock in the list.

  3. Click the Delete link in the Actions area to the right of the destination.

View Contact Information for Blocked Phone Numbers

  1. Search for the phone number in the Blocked Phones window.

  2. Check the Display Contacts check box.

  3. Click on the magnifying glass icon to the left of the contact to view the Contact page for the recipient.

Save a Local Copy of the Blocked Phones List

Simply click the CSV Download link for a CSV file containing the Blocked Phones list which you can open in a spreadsheet application such as Microsoft Excel.
