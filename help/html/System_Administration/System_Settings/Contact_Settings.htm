Contact Settings

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

The Contact Settings subtab contains options for the various job and destination types in the system. This subtab allows you to enter the information that Contact Manager users will see regarding job type and contact information when they are using their Contact Manager accounts. It also allows you to view, assign, or revoke Activation Codes for the Contact Manager, if this has been enabled on your account.

Contact Settings Home:

The above image shows the Contact Settings home which you will encounter when clicking on the Contact Settings subtab. The page is divided into two sections:

Job Types

  • Edit Job Types: Use this to edit job types that currently exist in the system.

  • Create New Job Types: Use this to create a new type of job in the system.

Destination Labels

  • Edit Phone Labels: Use this to edit the names of the different phone number fields in the system. For example, Phone 1 could be "Home Phone".

  • Edit Email Labels: This link allows you to edit the names of the different email fields in the system. For example, Email 1 could be "Work Email".

  • Edit SMS Labels: Use this to assign names to the different phone numbers that accept SMS messages.

 

Job Types

The links in this section allow you to edit settings regarding jobs in the system. You may edit existing job types or create a new type of job.

Edit Job Types

Clicking the Edit Job Types link will take you to the Job Type Manager, shown below. The Job Type Manager allows you to edit the information associated with existing job types. Each job type is organized by its priority in the system.

  • Name: This is the name of the job type in the system. It will be seen by users when creating or viewing jobs in the system. It will also be seen by Contact Manager users when they view the types of jobs that they can receive.

Note: The Emergency job type may not be edited.

  • Display Information: Enter a description for each job type which will be viewed by users. It will also be viewable by Contact Manager users, if this feature is enabled on your account.

  • Default Notification Setting: Check which contact methods the job type will use by default when contacting message recipients. When you mouse over each checkbox, the full name associated with each contact method will be displayed.

  • Delete: Clicking this button will delete the job type from the system. Emergency and High Priority job types may not be deleted.

 

Create New Job Type

Clicking the Create New Job Type link takes you to the Job Type Editor, shown below.

Note: New job types may only be General priority.

  • Name: Enter the name of the Job Type. This is how it will be seen by users creating jobs as well as by portal users.

  • Check if this is a Survey: Check this box if this job type is for surveys.

  • Display Information: Enter the description of the job type as you would like it to be seen by portal users.

  • Default Contact Preferences: Check the default methods that the job will use to contact recipients.

 

Destination Labels:

The links in this section allow you to name the different destination types in the system, for example, Phone 1 could be set to "Home Phone". The information entered in these pages will be displayed throughout the system.

Edit Phone Labels:

Clicking this link will take you to the page for editing Phone labels, shown below. Choose one of the system default labels from the drop down menu, or select Other and enter a custom label. Click the Done button when you have finished creating labels.

Edit Email Labels:

Clicking this link will take you to the page for editing Email labels, shown below. Choose one of the system default labels from the drop down menu, or select Other and enter a custom label. Click the Done button when you have finished creating labels.

Edit SMS Labels:

Clicking this link will take you to the page for editing SMS labels, shown below. Choose one of the system default labels from the drop down menu, or select Other and enter a custom label. Click the Done button when you have finished creating labels.
