Mapping App Categories to RSS Feeds

Important: Only those features that you have been granted access to will display. Additionally, features in this section of the help manual refer to integration with another product.

Overview

Push notifications are sent alongside RSS feed posts. Therefore, before you can begin to send push notifications to your custom mobile app, you'll need to map the categories in your app to RSS feed categories. If you haven't configured any RSS feeds yet, the Configuring Your RSS Feed help topic can show you how to do so.

Mapping Categories

  • Access the Feed Category editor by clicking the Admin tab followed by the Settings subtab. Then click the Feed Categories link in the Features section.

  • For each feed category, click the Map to CMA Categories button. This will take you to the mapping screen shown below.

  • Select the mobile app categories that should receive posts from this RSS feed.

  • Click Done to return to the Feed Categories screen.
