List Builder

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

What is List Builder?

List Builder helps you manage lists in the system. From List Builder, you can:

  • Preview, edit or delete lists.

  • Publish or subscribe to lists.

  • Create new lists using all of the advanced list building features.

Accessing List Builder

You can access List Builder by clicking the Broadcasts tab. By default, the Lists subtab is the first page you will see.

Preview, Edit or Delete Lists

Preview

The Preview action link to the right of each list in My Lists lets you view all of the recipients in the list.

Edit

Clicking the Edit action link to the right of a message in My Lists  will take you to the List Editor.

Delete

Click the Delete action link will delete the list.

Publishing or Subscribing to Lists

The Publish and Subscribe feature lets you share lists and messages with other users in the system. To learn more about, please view the Publish and Subscribe Overview.
