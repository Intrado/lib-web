Getting Started

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

This topic covers how to add individuals by uploading a list.

To use this tutorial, you should first follow the steps in Create a List. If you have done this, you will be in the List Editor, shown below.

Upload Types

You can upload two different types of lists into the system. You must use a CSV (comma separated value)  file for either type. You can easily create a CSV file using a spreadsheet application, such as Microsoft Excel.

Contact Data

Contact data CSV files contain complete contact information for your recipients with each column containing one field. For example: First name, Last name, email 1, email 2, phone 1, phone 2, etc.

  • From the List Editor, click the Upload List button in the Additional List Tools

  • Select the radio button labeled Contact Data.

  • Click Choose File to locate the file you would like to upload from your computer.

  • Click Preview. In the Upload Preview, use the menus to tell the system what type of data is in each field,  as shown in the following screenshot.

  • When you are done, click Save to return to the List Editor.

ID# Lookup

ID# Lookup CSV files contain only the ID numbers of recipients as they exist in your database. The system will use the ID number to locate the recipient's contact information already stored in your database.

  • From the List Editor, click the Upload List button in the Additional List Tools

  • Select the radio button labeled ID# Lookup.

  • Click Choose File to locate the file you would like to upload from your computer.

  • Click Preview to see the data for the recipients in your list, as shown in the following screenshot.

  • If your list looks correct, click Save to return to the List Editor.
