Add Groups of Contacts Using Rules

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

This topic covers how to add groups of contacts using rules.

To use this tutorial, you should first follow the steps in Create a List. If you have done this, you will be in the List Editor, shown below.

Note: Adding groups of contacts while building a list in the List Editor is just like adding groups of contacts when creating a new Broadcast from the Dashboard. The main difference is that the List Editor allows you to continue adding individuals to your list using other list building tools as well.

Build a List Using Rules

Rules let you set up filters to select subgroups of recipients from your database. Each rule is used to further define the group of people you want to include in your list. You can use as many rules as you like to define even more specific groups.

The easiest way to learn about building lists using rules is to look at some popular examples. The following sections will show you how to create some of the most popular lists using rules.

All School Student List

This list will contact all of the students in a particular school.

  1. Select School from the Field dropdown menu.

  2. Click the checkbox next to the desired school.

  3. Click Add to add the rule.

  4. Confirm the Total number of recipients is similar to the number of students you expected to include.

  5. Continue adding individuals or click Done if you're finished with your list.

Staff List

This is a list for contacting all Staff.

  1. Select Contact Type from the Field dropdown menu.

  2. Click the checkbox next to Staff.

  3. Click Add to add the rule.

  4. Confirm the Total number of recipients is similar to the number of students you expected to include.

  5. Continue adding individuals or click Done if you're finished with your list.

Grade Level List

This list uses multiple rules to first select all of the students from a school and filter those students by their grade level.

  1. Select Grade from the Select a Field dropdown menu.

  2. Click the checkbox next to the grade(s) you want to contact.

  3. Click Add to add the rule.

  4. Select School from the next Field dropdown menu.

  5. Click the checkbox next to the desired school.

  6. Click Add to add the rule.

  7. Confirm the Total number of recipients is similar to the number of students you expected to include.

  8. Continue adding individuals or click Done if you're finished with your list.
