Twitter

Important: This feature is only visible if your system includes the Social Media feature. For more information regarding the Social Media feature, please call 1-800-920-3897.

Connecting a Twitter Account

You will need to allow the SchoolMessenger Twitter app to have access to your account before you can post. You can do this from the Account link at the top of any page. You may also access these buttons as a step in MessageSender. You will only need to grant permission once. Your permission will last until you deliberately revoke it.

The image below shows the social media options on the Account Settings page, found at the Account link in the upper right corner of any screen.

To authorize a Twitter account, just click the "Connect to Twitter" button. You will be redirected to Twitter's web site where you may log in and grant our application permission to post. Once you grant permission, you will return back to our system.

Note: If you are currently logged into a Twitter account, this is the account you will authenticate. If it is not the correct account, please log out before you click "Connect to Twitter". If you've already connected to the account and it is incorrect, please read the section on "Changing Twitter Accounts".

Posting to Twitter

Including Twitter in your Broadcast is as simple as including any other destination type.

From the Dashboard

  1. Click the New Broadcast button.

  2. At the Message Content step, select the "Add Social" option and select Twitter.

  3. Enter your message

  4. Click Save Social Messages and continue adding content for the other destinations you wish to include.

Advanced Posting

If you would like to create your social media posts using advanced features such as the Page message type with embedded Page Media, you should create a message from Message Builder and use the features of the Message Editor to do so. If you've never created an Advanced Message, you should start by reading the Create a Message help topic. For details on adding RSS Feed content to an Advanced Message, you should read the Twitter Message Content help topic.

Removing a Twitter Account

To remove a Twitter account, go to the Account link in the upper right corner. Click on the "Disconnect from Twitter" button. This disconnects the system from your account. To completely deauthorize our application, you will need to log into your Twitter account and remove the application from your Settings page on Twitter's web site.

Changing Twitter Accounts

If you would like to change which Twitter account you will post to, simply follow the instructions to remove a Twitter account. After the account is removed, you will see the "Connect to Twitter" button again. Make sure you are logged out of the Twitter account, then click the "Connect to Twitter" button. Just log into the correct account and grant permission to post.
