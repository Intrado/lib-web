Facebook

Important: This feature is only visible if your system includes the Social Media feature. For more information regarding the Social Media feature, please call 1-800-920-3897.

Facebook is a great way to reach out to your contacts who are avid Facebook users. This feature allows you to connect to your Facebook account and post to your Wall or to any Facebook Pages where you are an administrator. Contacts who are your Facebook friends or who have liked your Facebook Page can then see your message when they log in.

Posting to Facebook

Including Facebook in your Broadcast is as simple as including any other destination type.

From the Dashboard

  1. Click the New Broadcast button.

  2. At the Message Content step, select the "Add Social" option and select Facebook.

  3. Enter your message.

  4. Click Save Social Messages and continue adding content for the other destinations you wish to include.

Advanced Posting

If you would like to create your social media posts using advanced features such as the Page message type with embedded Page Media, you should create a message from Message Builder and use the features of the Message Editor to do so. If you've never created an Advanced Message, you should start by reading the Create a Message help topic. For details on adding RSS Feed content to an Advanced Message, you should read the Facebook Message Content help topic.
