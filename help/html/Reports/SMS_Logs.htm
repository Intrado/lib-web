SMS Logs

This section contains information to help you interpret SMS Log reports.

Display Options

Guide to Report Results

This guide contains possible values which can be found in the Delivery Results column of the SMS Log report.

  • Sent: The SMS was sent to the recipient's cell phone.

  • Unsent: The SMS hasn't been sent to the recipient yet.

  • No SMS: There is no SMS cell phone in the database for the recipient.

The topmost part of the Phone Log contains field and output options for creating the report.

  • Display Fields: Additional fields which can be included in the report

  • Output Format: These buttons allow you to download your report as a CSV or PDF file.

 

Report Details

The lower half of the report contains the Report Details which shows each recipient called and the results of the call. It's sections include:

  • Job: The job that the recipient is associated with.

  • Submitted by: The user who submitted the job associated with the recipient.

  • ID#: The ID number of the recipient in the database.

  • First Name: The recipient's first name.

  • Last Name: The recipient's last name.

  • Message: The name of the message the recipient received. The icon next to the message displays the type of message sent; phone, email, or SMS.

  • Destination: The cell phone number that was used to reach the recipient.

  • Attempts: The number of attempts to reach this recipient.

  • Last Attempt: The last time that the system tried to contact the recipient.

  • Last Result: Shows the result of the delivery attempt. Results are explained in the Guide to Report Results in the blue box on this page.
