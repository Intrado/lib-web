Saving and Scheduling Reports

Many types of reports can be scheduled to automatically run at certain times. This is a great way to track regularly occurring data, like in the examples shown in this help topic.

To Schedule a Report

  1. Click the Reports tab.

  2. Choose the type of report you would like to schedule by clicking on it in the Select a Template area.

  3. Enter the report options that best describe your report.

  4. Click the Save/Schedule button.

  5. Enter a name for the report and a brief description, then check how often you would like the report to run.

  6. Click Save to schedule your report.

 

Examples of Types of Reports That Can Be Scheduled

  • Attendance Phone Log for an entire district: If your account's Access Profile allows you to view data for an entire district, you can schedule a Phone Log report displaying all of the absence notifications for your district at one time. This will show you everybody in your district that was absent for a certain day or range of days and the result of contacting them. For instance, the number recipients were successfully contacted with an attendance message. To create this report, you should use the following settings:

    • Search by Date and select the date or date range you're interested in viewing. If you're going to schedule this report to run repeatedly, you should select a relative date such as "Today" or "Week to date"

    • Select the "Attendance" Broadcast type.

  • All disconnected phone numbers for a school or district: If your account's Access Profile allows you to contact an entire school or district, you can schedule a Phone Log report displaying all of the calls which returned a value of "Disconnected" each week. To create this report, you should use the following settings:

    • Search by Date and select "Week to Date" or any date range you prefer.

    • If you want to limit your results to a single school, in the Criteria section, select "School" from the Field dropdown menu then check the box next to the school or schools you would like to view. If you want to view the results for the entire district and your profile allows you to do so, skip this step and simply filter by results.

    • Click "Disconnected" in the Filter by Results section.
