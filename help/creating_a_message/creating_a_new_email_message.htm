Creating a New Email Message

The My Email Messages section, in the middle of the Message Builder screen shown below, allows you to add a new message and also displays all previously saved messages.

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

To Create a New Email Message

Click the Create New Email Message button. This will take you to the Email Message Editor screen shown in the following image. If you plan to create this message in an alternate language be sure to indicate the language in the message name for easy identification when you create your notification job.

Writing Your Email:

  1. Enter a subject in the Subject field. This is what the recipient will see as the subject line of the email.

  2. Enter the name of the sender in the From Name field. This is who the recipient will see as the sender.

  3. Enter the sender's email address in the From Email field. This is the sender's email address as seen by the recipient.

  4. Enter the body of the email message in the Body field. You can include dynamic data fields at any point in your message. You may also use carriage returns for formatting.

  5. Click the Save button at the top or bottom of the screen when you are done. Your message will be saved and you will be returned to the Message Builder screen.  

 

Using Data Fields in Your Email:

Your email message can include dynamic data elements at any point in the message. For example, here is the dynamic message from the image above:

To the parents of <<First Name>> <<Last Name>>:

We're writing to inform you that your child, <<First Name>> <<Last Name>>, has achieved a <<GPA>> grade point average. <<First Name>>'s  exceptional grades have allowed him/her to achieve a place on our honor roll.

Congratulations!

The inserted data fields appear double bracketed <<like so>>. It is very important that the dynamic text not be edited. Altering the dynamic text may cause it to not display correctly.

How to Insert a Data Field:

  1. Place your cursor where you want the dynamic text to appear.

  2. Select the desired data field in the drop down menu to the right.

  3. Optional: Enter a default value recipients will see in the case where no data exists for a particular person.

  4. Click the Insert button.
