The Audio File Library

The Audio File Library is a menu for managing your personal audio recordings that you use when sending phone messages. The Audio File Library is accessible by clicking on the Notifications tab followed by the Messages subtab. The  Open Audio File Library button can then be seen on the right side of the My Phone Messages section of the Message Builder screen shown below.

Clicking the Audio File Library link will open the Audio File Library in a new window. The Audio File Library, shown in the image below, has a few important parts explained in greater detail in the following sections.

Audio Files:

This section shows you all of the audio files currently in the system. Clicking on the Play button will allow you to preview the audio file. The edit link in the Actions section of the Audio Files area will allow you to edit the name and description. Delete will remove the file from the system.

Call Me to Record:

Clicking on the Call Me to Record button at the top of the Audio File Library or on the Message Builder screen, will take you to the Call Me to Record screen shown here.

  1. Click the Call Me to Record button on the Message Builder page.

  2. Enter a name for your new message and the direct access phone number where you can be reached.

  3. Click Call Me to Record.

  4. You will receive a call from the system momentarily.

  5. Follow the prompts to record your message. To record more than one message, simply follow the prompts.

  6. When you are done, your new messages will appear in the Call Me window, as shown below, where you can assign them names.

  7. Click Save when you are done to save your messages and exit Call Me to Record.

Upload Audio File:

If you recorded a message using another audio editing program and would like to use it in a telephone message you can do so by first uploading the file to your Audio File Library. Currently supported formats are .wav, .aiff, and .au. Click on the Upload Audio File button at the top of the Audio File Library to use this feature. You will be taken to the screen shown below.

To use this feature:

  1. Enter a name and optional description for your audio file.

  2. Click the Browse button to search for the file on your computer.

  3. After selecting your file, click the Upload button.

Your audio file will appear in your Audio File Library list where you can preview it to insure that it plays back correctly.

 

When You Are Finished:

Click the Done button to close the Audio File Library.
