Activating by Phone

Important: This feature is only visible if your system includes the Contact Manager feature. The Contact Manager is a new feature from SchoolMessenger that allows your recipients to control the ways in which they may be contacted. For more information regarding Contact Manager, please call 1-800-527-5225.

Another way your Contact Manager users can activate their accounts is by phone. Simply check the Allow Activation via Phone checkbox in the Contact Manager Settings screen to enable this option.

When you allow users to activate their Contact Manager accounts by calling into the system, you will not need to generate Activation Codes. Users will only need to know their student's ID number.

If you choose to allow this option for account activation, your users will still need to go online to enter their ID number(s). When they create their Contact Manager account and log in for the first time, the system will guide them through account activation. The user will need to call the system from a phone number that is associated with the ID number they are attempting to activate.

Disabling Activation by Phone for Particular Users

If you would like to disable this feature for particular ID numbers, you may do so by unchecking the Allow Activation by Phone checkbox in the contact's details.

  1. Go to the System Tab and click on the Contacts subtab.

  2. Search for the contact.

  3. Click the icon next to the contact's name.

  4. Click the Edit button at the top or bottom of the screen.

  5. Uncheck the Allow Activation by Phone checkbox at the bottom of the contact's details, shown below.
