Generate an Individual Activation Code

Important: This screen is only visible if your system includes the Contact Manager feature. The Contact Manager is a new feature from SchoolMessenger that allows your recipients to control the ways in which they may be contacted. For more information regarding Contact Manager, please call 1-800-527-5225.

In addition to generating bulk activation codes for groups of Contact Manager users, you may also generate single codes for individuals. This feature is good for enabling new students or staff members to use the Contact Manager.

 

How to Generate an Individual Activation Code

  1. Click on the Admin Tab followed by the Contact Settings subtab.

  2. Click on the Activation Code Manager link.

 

  1. Click the By Person radio button in the Search section of the Activation Code Manager as shown above.

  2. Enter their ID number in the Person ID field and click Search.

  3. Click the Generate Activation Code button at the top or bottom of the screen.

  4. Make sure to notify the individual as to what their Activation Code is.
