Bulk Activation Code Management

Important: This feature is only visible if your system includes the Contact Manager feature. The Contact Manager is a new feature from SchoolMessenger that allows your recipients to control the ways in which they may be contacted. For more information regarding Contact Manager, please call 1-800-527-5225.

The Contact Manager feature allows message recipients to access and edit their contact information through a special web page. Contact Manager users may also select the ways in which they can be contacted. You may restrict which contact fields can be edited in the systemwide settings.

Each contact in the system may be associated with a Contact Manager account when the recipient creates an account and uses their Activation Code to associate a student. Activation Codes are assigned using the Activation Code Manager. The system will generate a CSV file containing the contacts and their respective Activation Codes which you will need to distribute.

Any setting that are changed through the Contact Manager may be overridden by system users whose access profiles allow them to do so by clicking the Details icon next to their ID# in the Search Results. Contact Manager accounts may also be disassociated from contacts if necessary. More information about this can be found in the Viewing and Editing Contact Details help topic.

To Access the Activation Code Manager

  1. Click on the System Tab followed by the Contacts subtab.

  2. Click on the Manage Activation Codes button.

 

The Activation Code Manager

 

  • Show All Contacts - Click this button to display all of the contacts in the system in the Search Results below.

  • Generate Activation Codes - Click this button to generate Activation Codes for the recipients in the Search Results section.

Contact Search:

Use the options in this section to select certain contacts in the system. You may then assign Activation Codes, view which recipients are currently Contact Manager users, or export a CSV file containing their Activation Codes for distribution.

  • Search - Select whether you would like to use search criteria or specific personal information to search by. Selecting criteria will display filtering options like those which are used for making lists. Selecting By Person will display a field where you must enter the person's ID number.

  • Display Fields - Use the checkboxes to display which fields should be displayed in the Search Results section.

  • Filter - Use the checkboxes to hide contacts who have Activation Codes which have not expired or contacts who are already active Contact Manager users. These filters allow you to assign Activation Codes to contacts who need them without overwriting codes that already exist.

  • Output Format - Click the CSV link to download a CSV file containing the information from the search results. You may use this list to distribute Activation Codes to your contacts. They will need this information to associate their contact information with their Contact Manager account.
