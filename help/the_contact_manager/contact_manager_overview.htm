Contact Manager Overview

The Contact Manager is an extra feature which enables your message recipients to control the ways in which they may be contacted and manage their contact information. For more information regarding this feature, please contact your SchoolMessenger representative at 1-800-527-5225.

If you already have Contact Manager on your account, use the following topics to help you manage your Contact Manager users.

  • Bulk Activation Code Management

  • Generate an Individual Activation Code

  • Contact Manager Settings
