Creating Notification Reports

A Notification Report is easy to create and shows you how successfully your message was delivered.

 

 
To Create a Notification Report:

  1. Click on the Reports tab. This will take you to the Report Builder screen.

  2. Enter a brief name and description for this report.

  3. In the Report Type area select the type of report you wish to generate. Reports can be based on relative date, a particular job, or for a date range.

  4. In the Sort Options section, choose the method you would like your data to be sorted by.

  5. In the Report Options section, you can select the parameters of the report. For example, the report for the image above would include every Elementary Attendance job sent by phone.

  6. If you wish to limit the results to a particular individual enter the person’s ID # in the filter options field located at the bottom of the window.

  7.  When you have completed your selections click the Create Report button to display the results.
