The Reports Tabs

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

Under the main Reports tab there are four sub-tabs. They are:

  • Report Builder - Use the form in this tab to create a notification report.

  • View Report - This tab displays the results of the last report that was created. The last report you created will display here until you log out or close your browser window.

  • Usage Stats - This tab graphically displays the total number of messages delivered systemwide based on users and jobs.

  • Call Distribution - There are two graphs displayed under this tab. The first graph displays the average call volume and results for the last seven days for all of the jobs in the system. The second graph displays the average call volume per day based on a systemwide average. This graph can be helpful in scheduling a job so as to avoid the highest volume times.
