Understanding Call Logs

Your report will show you how successful the system was at delivering your message. There are two main sections, the Report Summary and Report Details, each containing several fields which will be explained in detail below.

 

Report Summary

The top portion of the report displays the Report Summary which contains the results broken down by notification type as follows:

  • People: The number of people targeted to receive phone notifications included in the report.

  • Total: The total number of print or email notifications included in the report.

  • % Completed: The percent of the total that has been attempted thus far.

  • Delivered: The number of successful deliveries completed thus far.

  • Undelivered: The number of deliveries remaining.

  • Remaining: The number of people targeted to receive phone notifications who have not been attempted thus far.

  • Success Rate: The percentage of attempted calls that were successful.

  • Total Attempts: The total number of call attempts included in this report.

 

Report Details

The lower half of the report contains the Report Details which shows each recipient called and the results of the call. It's sections include:

  • ID: The ID number of the recipient in your database.

  • First Name: The recipient's first name.

  • Last Name: The recipient's last name.

  • Message: The name of the message the recipient received. The icon next to the message displays the type of message sent; phone, email, or print.

  • Destination: The phone number that was used to reach the recipient.

  • Attempts: The number of attempts to reach this recipient out of the number of attempts allowed. For example, a value of 2/3 would mean that the result displayed was for the second call attempt out of a possibility of 3 calls.

  • Last Result: Shows the result of the delivery attempt. Possible values are:

Answered: The recipient answered the phone and listened to the message.

Machine: The recipient's answering machine received the message.

Busy: The recipient's phone was busy.

No Answer: Nobody answered the phone.

Disconnect: This is typically a number which has been disconnected.

Failed: This indicates a temporary telco error and is usually resolved on the next attempt.

In Progress: The call is currently in progress or the results are currently being processed and more information will be available soon.
