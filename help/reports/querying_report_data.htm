Using the Report Builder

Specific data for archived or previously processed jobs can be easily queried using the Report Builder, shown below, found under the Reports tab.

 

 
To Create a Notification Summary Report:

  1. Click on the Reports tab. This will take you to the Report Builder.

  2. In the Select a Template area, click on the Notification Summary link. You will be taken to the following page.

  1. Select the options to create your report. You may select a particular job, date, or job type. If you are trying to create a report from a job that has been archived, check the "Show archived jobs" box.

  2. Click the View Report button to view your Notification Summary report. Clicking the Save/Schedule button will take you to the page shown below where you can save this report or schedule it to be run again in the future.

 

To Create a Survey Report

  1. Click on the Reports tab to access the Report Builder.

  2. In the Select a Template area, click on the Notification Summary link. You will be taken to the following page.

  1. Select the survey job from the drop down menu. Make sure that you click the checkbox for "Show archived jobs" if the survey you would like to view has been archived.

  2. Click the View Report button to view your Survey Report. If you would like to save the criteria you used for this report, click the Save/Schedule button.

 
To Create a Phone Log

  1. Click on the Reports tab to access the Report Builder.

  2. In the Select a Template area, click on the Phone Log link. You will be taken to the following page.

 

  1. Select the best options for the Phone Log you would like to view. For example, if you would like to create a report of all of the calls that resulted in No Answer, Failed, or Disconnected for a particular time period, then you would select "Date" in the Report Options section and the appropriate call results in the Filter by section. To select multiple items, hold down Ctrl while clicking on the items.

  2. Select any additional fields you would like to display in your Phone Log from the Display Fields section.

  3. Click View Report to view the results or Save/Schedule to save your options for later and/or schedule this report to happen again in the future.

 

To Create an Email Log


  1. Click on the Reports tab to access the Report Builder.

  2. In the Select a Template area, click on the Email Log link. You will be taken to the following page.

 

  1. Select the best options for the Email Log you would like to view.

  2. Select any additional fields you would like to display in your Email Log from the Display Fields section.

  3. Click View Report to view the results or Save/Schedule to save your options for later and/or schedule this report to happen again in the future.

 

To Create an SMS Log


  1. Click on the Reports tab to access the Report Builder.

  2. In the Select a Template area, click on the SMS Log link. You will be taken to the following page.

 

  1. Select the best options for the Phone Log you would like to view. For example, if you would like to only see when the call result was sent, you could select that in the Filter options. To select multiple items, hold down Ctrl while clicking on the items.

  2. Select any additional fields you would like to display in your SMS Log from the Display Fields section.

  3. Click View Report to view the results or Save/Schedule to save your options for later and/or schedule this report to happen again in the future.

 

To View the Contact History for a Particular Recipient

  1. Click on the Reports tab to access the Report Builder.

  2. In the Select a Template area, click on the Contact History link. You will be taken to the following page.

  1. Enter the ID number, phone number, or email address for the recipient you would like to report on.

  2. Select any additional filters in the Filter by section. For example, you may want to select a certain day or only the times that calls failed.

  3. Select any additional fields you would like to display from the Display Fields section.

  4. Click View Report to view the results.
