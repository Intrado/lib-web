Managing Your Account Settings

You may edit your account preferences by clicking on the Account link at the top of any screen. You will be taken to the page shown below.  

You may use this screen to change most of your account settings. Settings you may edit are:

First Name - Your first name.

Last Name - Your last name.

Username - The name you use to log into the system.

Password - Your password.

Telephone User ID# - The ID number that you may need use to log into the phone system for Remote Access.

Telephone PIN Code # - The PIN code that you use to log into the phone system.

Email(s) - Any email addresses you wish to use to receive reports. If you would like to include multiple email addresses, separate them with commas.

Phone - The phone number where you can be reached directly.

Default Delivery Window - The earliest and latest times you would like calls to be sent from your account by default. This will be additionally limited by your access profile.

Call Attempts - The number of times you would like calls from your account to be attempted by default. This will be additionally limited by your access profile.

Days to Run - The number of days you would like the system to keep your jobs active by default.

Caller ID - The phone number you would like recipients to see in their caller ID when the system is sending a call from your account.

Customize Theme - Check this if you would like to be able to change the Color Theme of the user interface for your account.

Color Theme - Select a custom Color Theme from the options in the dropdown menu.

Primary Color (in hex) - Enter the value of the color you would like for the text and shaded areas throughout the system. This needs to be the hex value for the color like you would use in HTML. You may also click on the palette icon to the right of the field to select a color from the menu.

Ratio of Primary to Background - This setting controls how much of your primary color is blended with the shaded areas in the interface. An example shaded area can be seen on the left side of the image above.
