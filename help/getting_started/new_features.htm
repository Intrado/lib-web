New Features

We've added several new features to enhance security, productivity, and user experience including:

Version 6.2

  • Improved Start Page - The start page now has a Quick Start section with buttons for faster access to the most popular features. For new users, there is a simplified start page with information to help them easily get started using the system.

  • Automatic Message Translation Powered by Google - Simple text-to-speech messages can now be automatically translated into several languages. This feature can be accessed through the Job Editor.

  • Enhanced language support - Now you can send messages in several languages! There are 14 languages available for text-to-speech phone calls and 33 languages for Email.

Languages which are now supported:

Text-to-Speech - English, Catalan (Valencian), Chinese (Mandarin), Dutch, Finnish, French, German, Greek, Italian, Polish, Portuguese, Russian, Spanish, and Swedish.

Email - English, Arabic, Bulgarian, Catalan (Valencian), Chinese (Manadarin), Croatian, Czech, Danish, Dutch, Finnish, French, German, Greek, Hebrew, Hindi, Indonesian, Italian, Japanese, Korean, Latvian, Lithuanian, Norwegian, Polish, Romanian, Russian, Serbian, Slovak, Slovenian, Spanish, Swedish, Ukrainian, and Vietnamese.

  • Call Back feature for retrieving messages - This new feature allows contacts to call back to hear missed messages. It can be ordered by contacting your sales representative.

  • Contact Manager Activation by Phone - Contact Manager users may now activate their account over the phone. This feature requires the user to call into the system from a phone associated with their account, but does not require an activation code.

 

New Features from Previous Releases

Version 6.1

  • Streamlined Job Editor - Creating a job with the Job Editor is easier than ever. The Job Editor now displays the most basic elements required for sending a job, greatly simplifying the process of sending a message. Advanced scheduling and multilingual options are still easily accessible as Advanced Options in the editor.

  • Group Type Data Imports - Group Fields allow you to map data in which a contact may have more than one value - such as dual campus enrollments, job titles or afterschool activities. For example, a student who is enrolled in two schools can now be associated with both schools without having their contact information duplicated between schools.

  • User Data Imports - Setting up and maintaining users can now be done with a data import, greatly simplifying user account setup and maintenance.

  • Enrollment Data Imports - Now you can import your students' scheduling data and restrict certain users to only access data for students who are associated with them - such as a student enrolled in a particular teacher's class. Enrollment fields allow you to connect students to staff and include additional enrollment information such as class names and class section codes. This feature makes it much easier for teachers to use the system to send messages to only the students enrolled in their classes.  

  • SMS Help and Opt Out – Text message recipients can receive additional information about the notification service or can opt out of receiving SMS messages by replying to an SMS message with the key words such as “HELP”, “STOP”, “UNSUBSCRIBE”, or “QUIT”.

  • Password Reset - If you forget your password, it's now possible to reset your password by clicking the Forgot Password link on the log in page. This feature requires your email address to be part of your contact information, as it will send you an email with a link to reset your password. This feature can also be used from the User Editor by clicking the Email Password Reset button.

 

Version 6.0

  • Flex Appliance - This feature allows you to use your own telco for day-to-day calling and take advantage of the high-volume capabilities of the ASP for emergencies. When using this feature, your database will be hosted on the SchoolMessenger ASP, reducing the need for maintenance and simplifying setup.

  • Multilingual Text-to-Speech Support - Now you can easily go between text-to-speech languages in the same message to enhance pronunciation and message clarity.

  • Message Confirmation - You may now request a confirmation of your message while you are setting up a job. Recipients may press 1 for "yes" or 2 for "no".

  • Message Playback from Reports - When looking at the Phone Log for a job in which the recipients were asked to record a message, you may now view and listen to their responses from within the report.

  • Phone Sequence added to Reports -  Instead of simply displaying the phone number that was called, Phone Logs additionally indicates which phone sequence number, for example, Phone 3, that the number is assigned to and the destination label.

  • Improved Contact Database Interface - When a contact's information has been edited in the Contact Information screen and the Override checkbox has been selected, the overridden destination will be protected during subsequent data imports. When this feature has been selected, a padlock icon will appear next to the locked information on the contact database screen.

  • Enhanced Job Editor - Now the Job Type description text displays in the Job Editor, next to the Job Type. Additionally, when you select the "Emergency" job type, the selection box will be outlined in red.

Version 5.2

  • Color Themes - Now you can choose a different color theme for the system's interface. This can be done as a system default as well as from each individual account.

  • List Upload - When uploading a list from your computer, you can now specify which fields the contact's data should go in. For example, you can specify if the phone number for each person should be mapped to Phone 1.

  • Improved Address Book - You may now control what types of messages are sent to which destination while entering new contacts manually.

Version 5.1

  • Delivery Point Management - This feature provides a fine degree of control to each individual recipient's contact preferences. Authorized users can now manage which types of notifications are sent to which contact point for each recipient individually. This means that if a recipient would like to receive Emergency messages at all of their phone numbers, but receive Survey messages only at their home phone, their preferences can now be set in the system. For more information on this feature, please read the Viewing and Editing Contact Details help topic.

  • Contact Manager - This feature gives your recipients control over the types of messages they receive and how they receive them. Individuals can edit their contact preferences through a web account. It also acts like a mailbox, giving them a place where they can review all of the messages that have been sent to them over the past 30 days. This optional feature is priced separately. Please contact your Account Manager for additional information and pricing.

  • Streamlined Notification Job Editor - Creating a new notification job is easier than ever with the Job Editor's simplified user interface and enhanced workflow. Simply check your job's message types, configure the applicable settings, and send!

  • Enhanced SMS Service - It is now possible to quickly send SMS messages directly from the Job Editor by simply entering the message as you are creating the job.

  • Email Attachment Support - Now users may add up to three attachments to their email notification campaigns.

  • Job Type Manager - Authorized administrators can use this feature to define new Job Types and destination labels. For example, Phone 1 can be labeled "Home", Phone 2 can be "Work", and so forth.

  • Invalid Login Attempt Lockout - This configurable lockout option enhances security by temporarily or permanently disabling user accounts after a specified number of unsuccessful login attempts. For example, if there are three login attempts using invalid information, the system could lock the user's account for five minutes. Please read the help topic, Managing Systemwide Settings, for more information.

  • Improved Field Mapping Tools - System Administrators can now enjoy a greater degree of control and flexibility when configuring data imports through the enhanced field mapping interface. Now, it is possible to define commonly used field formats and automatically apply data transformations to specific values. For example, school codes can be automatically converted to descriptive names. Please read the Import Manager help topic for more details.
