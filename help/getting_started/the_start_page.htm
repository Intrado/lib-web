The Start Page

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

EasyCall

The EasyCall button located in the EasyCall section provides a quick access tool for launching a basic phone notification. EasyCall lets you simply select an existing list to call, enter the direct dial phone number where you can be reached in order to record the message, and with a few simple clicks a call will be sent to tens, hundreds, or even thousands of recipients.

My Active Calls

The graph displayed in this area shows the current results of all active jobs on your account. To view a graph for a specific job that is running, click the Monitor link next to the job in the My Active and Pending Jobs section.

My Active and Pending Notifications

The My Active and Pending Jobs section of the Start page displays a list of the notification jobs which are currently running or waiting to be run in your account. Active and pending jobs may be viewed and edited, and the jobs themselves may be canceled from this area if desired. Simply clicking the Report link next to the name of a job that is currently running allows you to access the current report results for that job. The Monitor link will open a small window that displays the current results of the job as a graph.

My Completed Notifications

This section displays information about some of the most recently completed notification jobs from your account. To the right of each job you will find five links. The Edit link allows you to edit the name and description of the job. The Report link takes you to the report results for the job. Graph shows the delivery results in a graph. The Responses link takes you to the Responses Tab for the job where, if you allowed recipients to leave responses, you can listen to the recordings. Lastly, Archive will archive the results so they are no longer visible in any completed jobs lists. The job's results can still be accessed by creating a custom report.
