The Start Page

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

New Users

The first time you log into your account, you will see a simple Start Page with information to help you become acquainted with the system and start sending calls.

Experienced Users

Once you've made a list, the normal Start Page will be displayed. It has several important features designed to help you quickly get your message sent and see your job's progress at a glance.

 

Quick Start

  • Basic - The EasyCall button located in the Quick Start section provides a quick access tool for launching a basic phone notification. EasyCall lets you simply select an existing list to call, enter the direct dial phone number where you can be reached in order to record the message, and with a few simple clicks a call will be sent to tens, hundreds, or even thousands of recipients.

  • Advanced - The Create New Job button takes you straight to the Job Editor where you can quickly assemble a job from the lists and messages you have already created.

  • List - The Create New List button takes you straight to the List Editor where you can build a list for your jobs.

My Active Calls

The graph displayed in this area shows the current results of all active jobs on your account. To view a graph for a specific job that is running, click the Monitor link next to the job in the My Active and Pending Jobs section. When there are no active jobs, this section will be closed.

My Jobs

The My Jobs section of the Start page displays a list of the notification jobs which are currently running or waiting to be run in your account. Jobs may be viewed and edited, and the jobs themselves may be canceled from this area if desired. Simply clicking the Report link next to the name of a job that is currently running allows you to access the current report results for that job. The Monitor link will open a small window that displays the current results of the job as a graph.

My Completed Jobs

This section displays information about some of the most recently completed jobs from your account. To the right of each job you will find five links. The Edit link allows you to edit the name and description of the job. The Report link takes you to the report results for the job. Graph shows the delivery results in a graph. The Responses link takes you to the Responses Tab for the job where, if you allowed recipients to leave responses, you can listen to the recordings. Lastly, Archive will archive the results so they are no longer visible in any completed jobs lists. The job's results can still be accessed by creating a custom report.

My Surveys

The My Surveys section of the Start page displays a list of the survey jobs which are currently running or waiting to be run in your account. Active and pending surveys may be viewed and edited, and the jobs themselves may be canceled from this area if desired. Simply clicking the Report link next to the name of a job that is currently running allows you to access the current report results for that job. The Monitor link will open a small window that displays the current results of the job as a graph.
