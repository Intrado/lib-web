The Mechanism Tab

If you click on the Mechanism tab you will see two subtabs, Settings and Plug-In.

Important: The settings you see have been established by your SchoolMessenger representative and are designed to be maintenance free. Editing these settings could cause your system to not function properly.

The Settings tab has several sections:  

  • Type – Tells what type of delivery mechanism you are looking at. This value depends on which delivery mechanism you have logged into.

  • Name – This is the name of the delivery mechanism.

  • Description – This space is for an optional description of the delivery mechanism for your personal notes

  • Application Server URL – This tells the Delivery Mechanism where to find the SchoolMessenger CommSuite application.

  • Username, Password, and Confirm Password  - These spaces are for setting the username and password for logging into the Delivery Mechanism screen.

  • Media Cache Age – This is the maximum duration that an item will stay in the cache before being deleted if not used. It should be at least one minute.

  • Media Cache Size – The number of megabytes used for the cache. Anywhere between 100 megabytes and 1 gigabyte should suffice.

  • Media Cache Path – This is the location where all the audio files and text-to-speech renderings will reside. This is usually configured by Reliance and should not be edited.

  • Enabled – The most important thing on the page. If unchecked the resources will gracefully shut down. It will not hang up in the middle of a call. The delivery mechanism can be enabled again only when all of the resources are fully shut down.

The Plug-In tab is used to configure the type of delivery mechanism you have. Its settings are configured by your SchoolMessenger representative. Do not edit these settings. Doing so will cause your system to not function properly.
