The List Editor Screen

The List Editor screen seen in the image below is where you can configure your recipient list. You can get to this screen by either creating a new list or editing an existing list. The sections below will describe the various parts of the List Editor screen in greater detail.

The Refresh, Search, Preview, and Done Buttons:

These buttons are conveniently located at both the top and bottom of the left side of the screen.

  • Refresh: Use this button to refresh the screen, allowing you to see certain changes reflected in your List Editor screen.

  • Search: This button opens up the List Search page where you can add or remove people from your list with a simple search utility.

  • Preview: Use this button to preview your list. In the Preview screen you can add or exclude individuals by simply clicking the check box next to their name.

  • Done: This button will save your list and return you to the List Builder screen.

 

List Information:

This section of the List Editor screen allows you to add or change the name and description of your list. It also shows how many people are currently in your list. If you change the title of your list, clicking the refresh button will reflect that change in the title displayed at the top of the screen.

List Content:

The List Content section of the List Editor screen is where most list configuration takes place. The following is a brief overview of each part of the List Content section.

Rules: This is the area where you can set rules to create dynamically updating lists.

Additions: You can view manually added contacts to your list here.

Skip: This area shows individuals who have been excluded from your list. Clicking the check box next to their name will include them in your list again.

Search Database: Click this button to add people by searching the database.

Manual Add: This is where individuals can be added to your list by manually entering their information. The contact information will be automatically stored in your address book for future use.

Open Address Book: Click this button to add people from your address book to this list.

Upload List: This feature allows you to upload a list of contacts from applications such as Excel by saving them as a CSV file.
