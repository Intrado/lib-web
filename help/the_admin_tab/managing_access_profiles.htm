Managing Access Profiles

Access Profiles allow the system administrator to control access to certain application features by defining profiles and assigning them to users. The Security tab allows the system administrator to create, edit, and delete access security profiles. Prior to creating user accounts access profiles need to be defined so that they can be assigned to individual users.

The Access Profiles screen seen below can be accessed by clicking on the Admin tab followed by the Security subtab.

Adding a New Access Profile:

  1. Click the Create New Access Profile button. You will be taken to the Edit Access Profile screen.

  2. Enter a brief name and description for the profile.

  3. Configure the profile.

  4. Click the Save button when you are done.

Editing an Existing Profile:

  1. Click the Edit link to the right of the profile you want to edit.

  2. Edit the configuration in the Edit Access Profile screen.

  3. Click the Save button when you are done.

Deleting an Existing Profile:

  1. Click the Delete link to the right of the profile you want to remove.

Note: Access Profiles that are in use cannot be deleted. Deleting cannot be undone.

 

The Edit Access Profile Screen:

The Edit Access Profile screen shown below allows you to configure your Access Profiles, controlling what certain users will be able to do within the system. It has several sections which will be explained below.

Login:

  • Log into the system via the web - Check this option to allow users with this profile to access the system via their computer's web browser.

  • Log into the system via the phone - Check this option to allow users to log in by calling and entering their Personal Identification Number.

  • Edit person account information - Check this option to allow users with this profile to edit their personal account information such as their name and password.

Start Page:

  • View Start Page Job Statistics - Check this option to allow users with this Access Profile to see their recent and completed jobs on their start page.

  • View Start Page Shortcuts - Check this option to allow users to see the Shortcuts section of their start page.

  • Use EasyCall option - Check this option to give users with this profile the ability to use EasyCall for their callouts.

Messages:

  • Send phone calls - Check this option to allow users with this profile to create phone jobs. The drop down menus to the right allow you to set the time range in which the calls can run as well as the maximum number of times the system can try to call each recipient. The user will be able to alter these values only to values which fall within this range.

  • Send emails - This allows the user to create email jobs.

  • Send multi-language messages - Check this to enable the Multilingual Job Options feature on all enabled job types.

Lists:

  • Create/Edit Lists - Check this to allow users to create or edit lists.

Data:

  • Allow user to build lists and construct messages using the following fields - Check this option and select data fields from the list to the right to give the user the ability to create dynamic lists and messages based on the fields they are given access to. Select which fields the user may use in the menu to the right. Hold down the Ctrl key (Command key for Mac users) while selecting to choose multiple fields.

Jobs:

  • Create Repeating Jobs - This gives the user the ability to create repeating jobs such as attendance.

  • Set the job caller ID - This gives the user the ability to enter information recipients will see on their caller ID.

  • Maximum number of days for which users can schedule a job to run - This sets the maximum number of days a user can schedule a job to run.

Reports:

  • Create Reports - Check this box to give the user the ability to create reports.

Security:

  • Manage User Accounts - Check this box to allow users to create, edit, and delete User Accounts.

  • Manage Access Profiles - Check this box to allow users to create, edit, and delete Access Profiles.

  • Manage Systemwide Settings - Check this box to allow users to edit Global System Settings.

Contacts:

  • View Contacts - Check this box to allow users to view contacts stored in the system's contact list.

Systemwide View:

  • View systemwide reports - Allows the user to see systemwide information when they view the completed jobs page or when they generate a report from the Reports tab.

  • View all active jobs - Select to allow the user to view all jobs currently running in the system.

  • View all completed jobs - Select to allow the user to view all completed jobs in the system.

  • View all repeating jobs - Select to allow the user to view all repeating jobs currently scheduled in the system.

  • Manage systemwide jobs - Select to allow the user to edit or cancel any job in the system.

  • Manage Metadata - Select to allow the user to configure new fields in the system's database for data imports.

  • Manage Tasks - Select to allow the user to configure, create, and delete data imports in the system.

  • Access systemwide blocked numbers list - Select to allow the user to view the systemwide blocked number list. If you select this option, you may choose the amount of access the user will have from the options below. The options are:

View numbers only - Allows the user to view the phone numbers, but not add or remove any numbers.

Add/Delete Own Numbers - Allows the user to add or delete phone numbers, but they cannot delete numbers entered by any other user.

Add/Delete All Numbers - Allows the user unlimited ability to add or delete phone numbers.
