Managing Global System Settings

The Settings tab allows the system administrator to edit and manage certain global system options explained below. After you have adjusted the Global System Settings, click the Save button at the top or bottom of the screen to save your changes.

Job Type/Priorities:

  • Add a new type of job using the field at the bottom of the section and click the Add button to add it to the list of available job types. You can adjust the job's priority in the system by click the up or down arrows to the right of the job type. The priority level specifies which type of job will take precedence in the system. For example, in the image above if a user sent out a District Emergency job while an Elementary Attendance job was running, the District Emergency job would run and the Elementary Attendance would be paused.

  • Clicking the Edit link to the right of any job type will allow you to delete that type from the system.

  • Clicking the Delete link to the right of any job type will allow you to delete that type from the system. Deleting cannot be undone.

Note: Changing a job type's priority will not affect a job that is already in progress.

 

Retry Settings:

  • Use the drop down menu to select the amount of time the system should wait in between attempts to call an unanswered phone number.

 

Disable Repeating Jobs:

  • Click this checkbox to prevent the system from performing any of its scheduled repeating jobs. Unchecking this box will allow the repeating jobs to resume at their next scheduled time.

 

Default Caller ID Number:

  • Enter the phone number you would like to display in the recipient's Caller ID.

 

Systemwide Alert Message:

  • Text entered here will appear as an alert message for all users in the system. The alert message appears as black text in a red box on every page.
