Step-by-Step: Create a List

Before you can send any type of notification you must create a list. This list will determine who receives the message you send. Lists can either be dynamic, static, or a combination of both.

  • Dynamic List -  A list that is based on rules and that automatically updates as the data in the systemâ€™s database changes. For example, to create a list that only includes the 12th grade students you could simply use the rule 'Grade is 12'.

  • Static List - A list built either by selecting individuals from the system database or by manually typing individuals into the list. The people included on this list will not change unless you manually edit the list. For example, a list of PTA parents could be created by manually selecting each PTA parent that you wish to include on the list.

This tutorial walks beginning users through the steps of creating a list. Most users will find that after creating a few simple lists they can create any number of complex lists without further assistance.

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

Log In

Enter your user name and password at the login page and click the Sign In button. This will bring you to the Start page.

 

Create a New List

  1. Click the Notifications tab and then the Lists tab. This will open your List Builder page (See Figure 1).

  2. Click the Create New List button. This will take you to the List Editor screen.

  3. Enter a brief name and description of your list in the fields at the top of the List Editor screen.

 

Figure 1

Adding Rules to a List

  1. To create a dynamic list, start by selecting the field that you wish to base the list on. A list of available data items for that field will display when you click on the Select a Field drop down menu (See Figure 2).

  2. Select the comparison operator to use, and then select the value(s) to include with this rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking. For example, "Grade is 12" will create a list of all 12th graders.

  3. Click the Add button to add this rule to the list. Add additional rules to this list as required (See Figure 2).

  4. When you have added all of the rules you want to include, check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. See the Previewing and Saving Your List section below if you are finished with this list.

Searching for Individuals to Add

  1. To search the database for individuals to add to this list click the Search button. This will open the List Search window where you can enter rules by which to search (See Figure 3).

  1. Select the search criteria you wish to use. The matching individuals will automatically display (See Figure 4).

  2. When you locate an individual that you want to include on the list, simply check the box to the left of their name. Any person with a + symbol in the checkbox will be included (See Figure 4). When you are finished adding individuals click the Done button to return to the list editor page.

  3. Verify that the individual(s) you selected appear in the Additions section of the list editor. You can remove an individual at any time by unchecking their name and then clicking the Refresh button.

  4. Check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. See the Previewing and Saving Your List section below if you are finished with this list.

Manually Adding Contacts to Your List

Note: By default this automatically adds the person you entered to your Address Book. This allows you to easily add them to other lists you create without retyping their contact info.

  1. To manually add a person to your list, enter the contact information in the Manual Add section of the list editor, and then click the Add button (See Figure 5).

  2. Verify that the individual(s) you manually added appear in the Additions section of the list editor. You can remove an individual at any time by unchecking their name and then clicking the Refresh button.

  3. Check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. See the Previewing and Saving Your List section below if you are finished with this list.
