Step-by-Step: Send a Job with EasyCall

This guide will walk you through the quickest and easiest way to send a call out. There are two basic steps to getting your phone message sent.

  1. Create a List

  2. Record and Send Your Message with EasyCall

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

 

Create a List

Note: If you have already created the list that you wish to use skip this section. If you want to use a list building method not included in this guide please refer to the Creating a List main help section.

Before you can send any type of notification you must create a list. This list will determine who receives the message you send. Lists can either be dynamic, static, or a combination of both.

  • Dynamic List - A list that is based on rules and that automatically updates as the data in the system’s database changes. For example, to create a list that only includes the 12th grade students you could simply use the rule ’Grade is 12'. This is the best way to call large groups, such as an entire school.

  • Static List &endash; A list built either by selecting individuals from the system database or by manually typing individuals into the list. The people included on this list will not change unless you manually edit the list. For example, a list of PTA parents could be created by manually selecting each PTA parent that you wish to include on the list. This is the best way to add an individual, such as yourself, to the recipient list.

To Create a List

  1. Click the Notifications tab and then the Lists tab. This will open your List Builder page (See Figure 1).

  2. Click the Create New List button. This will open the List Editor.

  3. Enter a name and description for your list in the fields at the top of the List Editor screen.

Adding Rules to a List (creating a dynamic list)
Use this section to create a list using fields from your database that dynamically updates every time you use it.

  1. To create a dynamic list, start by selecting the field that you wish to base the list on. A list of available data items for that field will display when you click on the Select a Field drop down menu.

  2. Select the comparison operator to use, and then select the value(s) to include with this rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking. For example, ”Grade is 12” will create a list of all 12th graders.

  3. Click the Add button to add this rule to the list. Add additional rules to this list as required (See Figure 2).

  4. When you have added all of the rules you want to include, check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. Skip to the Previewing and Saving Your List if you are finished with this list.

Manually Adding People to Your List (creating a static list)
Use this section to add individuals to your list. This is a great way to include yourself in your list so that you are called along with everyone else when your job runs.

You can add a new person by clicking the Enter Contacts button in the Manual Add section of the List Editor screen.

 

To Manually Add Individuals To Your List:

  1. In the List Editor screen, click the Enter Contacts button in the Manual Add section. You will be taken to the Enter Contact Information screen shown above.

  2. To automatically add the individual's contact information to your Address Book, make sure the "Save to My Address Book" option at the bottom of the screen is checked.

  3. Click the Save & Add Another button in the upper or lower left corner to add the individual to your list and enter contact information for another person. If you don't need to add another person, proceed to step 4.

  4. When your list is complete, click the Save & Done button, located in the upper or lower left corners to save your list, or continue to add individuals using a different method.

Previewing and Saving Your List

  1. When you've finished adding people to your list, note the current total number of people in your list displayed near the top of the editor window. Click the Preview button to display the current list. You can remove any individuals whom you don’t want to contact by unchecking the box to the left of their name.

  2. Click Done when finished.

  3. Click Save to save this list and return to your List Builder page.

 

Record and Send Your Message with EasyCall

Note: If you need to send a message with dynamic content, please refer to the Creating an Advanced Phone Message online help topic.

The EasyCall feature can be found on your Start Page. To create a new EasyCall job, just click the EasyCall button.

After you click the Easy Call button the EasyCall window will pop up.

Using EasyCall:

  1. Select the type of job you are sending.

  2. Select your list.

  3. Enter the direct access phone number where you are located.

  4. Click the Call Me to Record button. While the system is calling you, you will see the following screen.

  1. When the system calls you, just follow the prompts to complete your recording. You may also record your message in different languages. After you finish and hang up, you will see the following screen.

  1. Here you may confirm that your message sounds correct and submit your job by clicking the Submit Job button.

  2. You may view the status of your job on the Notification Jobs page by clicking on the Notifications tab followed by the Jobs tab.
