Step-by-Step: Send an Email Job

In addition to sending phone notifications SchoolMessenger has numerous features for sending simple or complex multi-modal emails to any number of recipients. Emails can be sent in conjunction with a phone notification job, or a job can simply consist of only emails.

There are three basic steps to sending an email job which this guide will walk you through:

  1. Create a List

  2. Create an Email Message

  3. Send Your Job

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

 

Create a List

Note: If you have already created the list that you wish to use skip this section. If you want to use a list building method not included in this guide please refer to the Creating a List main help section.

Before you can send any type of notification you must create a list. This list will determine who receives the message you send. Lists can either be dynamic, static, or a combination of both.

  • Dynamic List - A list that is based on rules and that automatically updates as the data in the system’s database changes. For example, to create a list that only includes the 12th grade students you could simply use the rule 'Grade is 12'. This is the best way to call large groups, such as an entire school.

  • Static List - A list built either by selecting individuals from the system database or by manually typing individuals into the list. The people included on this list will not change unless you manually edit the list. For example, a list of PTA parents could be created by manually selecting each PTA parent that you wish to include on the list. This is the best way to add an individual, such as yourself, to the recipient list.

To Create a List

  1. Click the Notifications tab and then the Lists tab. This will open your List Builder page (See Figure 1).

  2. Click the Create New List button. This will open the List Editor.

  3. Enter a name and description for your list in the fields at the top of the List Editor screen.

Adding Rules to a List (creating a dynamic list)
Use this section to create a list using fields from your database that dynamically updates every time you use it.

  1. To create a dynamic list, start by selecting the field that you wish to base the list on. A list of available data items for that field will display when you click on the Select a Field drop down menu.

  2. Select the comparison operator to use, and then select the value(s) to include with this rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking. For example, 'Grade is 12' will create a list of all 12th graders.

  3. Click the Add button to add this rule to the list. Add additional rules to this list as required (See Figure 2).

  4. When you have added all of the rules you want to include, check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. Skip to the Previewing and Saving Your List if you are finished with this list.

Figure 2

Manually Adding People to Your List (creating a static list)
Use this section to add individuals to your list. This is a great way to include yourself in your list so that you are called along with everyone else when your job runs.

Note: By default this automatically adds the person you entered to your Address Book. This allows you to easily add them to other lists you create without retyping their contact info.

  1. To manually add a person to your list, enter the contact information in the Manual Add section of the list editor, and then click the Add button (See Figure 3).

  2. Verify that the individual(s) you manually added appear in the Additions section of the list editor. You can remove an individual at any time by unchecking his/her name and then clicking the Refresh button.

  3. Check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. See the Previewing and Saving Your List section if you are finished with this list.

Figure 3

Previewing and Saving Your List

  1. When you've finished adding people to your list, note the current total number of people in your list displayed near the top of the editor window. Click the Preview button to display the current list. You can remove any individuals whom you don’t want to contact by unchecking the box to the left of their name.

  2. Click Done when finished.

  3. Click Save to save this list and return to your List Builder page.

 

Create an Email Message

Note: If you already have an email message created that you wish to use skip this section.

Figure 5

  1. Click the Messages tab. This will open your Message & Survey Template Builder page (See Figure 5).

  2. In the My Email Messages area click the Create New Email Message button. This will open the message editor.

Figure 6

  1. Enter a brief name and description for your message in the Message Information section.

  2. Enter the subject of the email just as you would for any typical email that you send. (See Figure 6).

  3. Enter the name and email address that you want the email to display as being from (See Figure 6).

  4. In the Body section enter the text of the message that you wish to send. This can include text, dynamic data inserts and carriage returns for formatting (See Figure 6).

  5. Click Save to save this message and return to your Message Builder page.

 

Send Your Job

Figure 7

  1. Click the Jobs tab to open the Notification Jobs page (See Figure 7).

  2. Click the Create New Job button in the My Active and Pending Jobs section (See Figure 7). This will open the Job Editor where you can enter a name and description for your job.

Figure 8

  1. Select the type of job you are sending.

  2. Select the list that you wish to contact (See Figure 8).

  3. Enter the start and end dates for the job. Most standard date formats are recognized – for example, 02/25/05 or February 25th, 2005 (See Figure 8).

Figure 9

  1. Select the Send Emails checkbox, and then select the Default Message that you wish to send (See Figure 10). (Note: Unless multilingual message options are specified this will be the message that is sent to all recipients.)

  2. If your system has multilingual messaging options enabled you can specify alternate messages to be delivered for people with particular language preference settings by selecting specific language and message combinations.

  3. Click the Submit Job button to submit your job for processing.

  4. From the Start tab or from the Jobs tab you can view the status of this job. You can also get an up-to-the-minute report of the job progress by clicking the Report link located to the right of the Job name.
