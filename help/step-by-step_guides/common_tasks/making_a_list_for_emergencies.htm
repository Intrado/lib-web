Making a List for Emergencies

This tutorial will walk you through the steps to create a list which will call an entire school. This is not only very useful in the event of an emergency, but a list which can be used for general announcements as well.

Create a List for Calling All the Students in a School

  1. Click the Create New List button on your Start Page to go to the List Editor.

  2. Enter a descriptive name for your list such as "All Students".

  3. Click Save to start building your list.

  4. In the List Content section, click the dropdown menu in the rules for "School Name".

  5. Leave the middle dropdown at "is" and choose the school name from the next dropdown menu. Your List Editor should look like the following image.

Note: If you want this list to call more than one school, hold the Ctrl key (Command key on the Mac) and select multiple schools.

  2. Click the Preview button in the List Information section to preview your list.

  3. When you are finished previewing click the Done button in the upper or lower left corners.

  4. Click the Done button to save your list and exit the List Editor.

Now you have a list which can be used for calling an entire school. To make lists which include smaller groups from a school, just make another list like this one and add additional rules. For example, adding another rule for "Grade is 6" to "School is Anytown ES" will restrict the list to calling only the 6th graders in this school.

Remember: Lists can be reused! It's not necessary to create a new list for every job. You can use the list created in this tutorial for every schoolwide general announcement as well as for emergency calls.
