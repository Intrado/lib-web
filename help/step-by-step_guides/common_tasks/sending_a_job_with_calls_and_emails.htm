Sending a Job with Calls and Emails

You can include emails and phone calls in a single job. This tutorial assumes that you have already created a list. If you have not created a list yet, you will need to do so before proceeding.

If you use text-to-speech for your message, you can create both the phone message and email in the Job Editor. You can also pre-record a phone message or pre-format an email using the features in the Message tab.

Building the Job

    1. From your Start Page, click the Create New Job button.

    2. Click the Phone icon to open up the Phone options and the Email icon to open the Email options.

    3. Fill in the appropriate information in the Settings section, including selecting the list or lists you would like to send your message to.

    4. In the Phone section, choose the option that matches the message you are trying to send.

      • Select a Message: When this option is selected, you may choose a message which you've already created.

      • Create a text-to-speech message: Select this option if you would like to type a simple message for the job.

    1. In the Email section, choose the best option for your message.

      • Select a Message: When this option is selected, you may choose a message which you've already created.

      • Create a Message: Choose this option if you would like to create a new message.

    1. When everything looks correct, click the Proceed to Confirmation button. Click the Save for Later button if you would prefer to send this job at another time.
