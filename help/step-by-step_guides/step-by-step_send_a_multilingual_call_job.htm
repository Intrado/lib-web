Step-by-Step:Send a Multilingual Call Job

In addition to sending phone messages to recipients in English you can also create a second message in an alternate language and specify that it be delivered to those individuals in the system who have that language preference specified in their contact data.

This tutorial walks beginning users through the steps of creating a list, creating both an English and Spanish phone message and then using those messages in a call job launched from the system’s web interface. Most users will likely find that after sending a few multilingual phone notifications the steps to perform this type of callout can be performed without assistance.

Any number of other language messages can be created and used in targeted notifications by following the same steps. The only restriction to the number of different language messages that can be sent will be the number of language definitions that have been configured for your system by the system administrator.

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

 

Create a List

Note: If you have already created the list that you wish to use skip this section. If you want to use a list building method not included in this guide please refer to the Creating a List main help section.

Before you can send any type of notification you must create a list. This list will determine who receives the message you send. Lists can either be dynamic, static, or a combination of both.

  • Dynamic List - A list that is based on rules and that automatically updates as the data in the system’s database changes. For example, to create a list that only includes the 12th grade students you could simply use the rule ‘Grade is 12'. This is the best way to call large groups, such as an entire school.

  • Static List – A list built either by selecting individuals from the system database or by manually typing individuals into the list. The people included on this list will not change unless you manually edit the list. For example, a list of PTA parents could be created by manually selecting each PTA parent that you wish to include on the list. This is the best way to add an individual, such as yourself, to the recipient list.

To Create a List

  1. Click the Notifications tab and then the Lists tab. This will open your List Builder page (See Figure 1).

  2. Click the Create New List button. This will open the List Editor.

  3. Enter a name and description for your list in the fields at the top of the List Editor screen.

Adding Rules to a List (creating a dynamic list)
Use this section to create a list using fields from your database that dynamically updates every time you use it.

  1. To create a dynamic list, start by selecting the field that you wish to base the list on. A list of available data items for that field will display when you click on the Select a Field drop down menu.

  2. Select the comparison operator to use, and then select the value(s) to include with this rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking. For example, “Grade is 12” will create a list of all 12th graders.

  3. Click the Add button to add this rule to the list. Add additional rules to this list as required (See Figure 2).

  4. When you have added all of the rules you want to include, check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. Skip to the Previewing and Saving Your List if you are finished with this list.

Figure 2

Manually Adding People to Your List (creating a static list)
Use this section to add individuals to your list. This is a great way to include yourself in your list so that you are called along with everyone else when your job runs.

Note: By default this automatically adds the person you entered to your Address Book. This allows you to easily add them to other lists you create without retyping their contact info.

  1. To manually add a person to your list, enter the contact information in the Manual Add section of the list editor, and then click the Add button (See Figure 3).

  2. Verify that the individual(s) you manually added appear in the Additions section of the list editor. You can remove an individual at any time by unchecking his/her name and then clicking the Refresh button.

  3. Check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. See the Previewing and Saving Your List section if you are finished with this list.

Figure 3

Previewing and Saving Your List

  1. When you've finished adding people to your list, note the current total number of people in your list displayed near the top of the editor window. Click the Preview button to display the current list. You can remove any individuals whom you don’t want to contact by unchecking the box to the left of their name.

  2. Click Done when finished.

  3. Click Save to save this list and return to your List Builder page.

 

Sending a Multilingual Call

This section will outline two optional ways to send a Multilingual call.

  • Option A - The method outlined in this section will allow you to quickly send a Multilingual call using EasyCall. We will create a list of Spanish speakers and a list of English speakers, then use EasyCall to record and send the appropriate message to each. Note: This method is very easy, but creates two jobs and therefore two reports.

  • Option B – The method outlined in this section will allow you to create only one list and use the Job Builder to specify which message the recipients will receive. Note: This method is a little more involved, but creates only one job with one report.

 

Option A - Creating Language Specific Lists

1. Create a new list following the steps outlined in the Create a New List section. Make sure to specify which language your list contains (ex. “4th Grade Parents-Spanish” and “4th Grade Parents-English”)

2. Define your list using rules as outlined in the Adding Rules to a List section. This time, however, add the rule “Language is Spanish” or “Language is English” as appropriate.

3. Add any individuals not covered by rules, such as yourself if you desire, or proceed to the next step.

4. Verify that your list looks correct and click the Save button.

5. For each list, follow the instructions in the Creating a Job with EasyCall section. Make sure that you include the language in the name of the job. (ex. “No School Spanish” and “No School English”)

 

Option B – Use the Job Builder to Specify Language

1. Create a new list following the steps outlined in the Create a New List section.

2. Define your list using rules as outlined in the Adding Rules to a List section.

3. Click on the Notifications tab followed by the Messages tab. This will take you to the Message Builder screen (see figure 7).

Before continuing, it is important to note that there are three ways to create your message content. The following sections illustrate how to record a message over the phone. For advanced message creation options such as uploading audio files or using text-to-speech, please see the online help.

Create a Message Using the Telephone to Record

1. Click the Notifications tab followed by the Messages tab to get to the Message Builder screen (see figure 7).

2. Click the Open Audio File Library button.

3. When the Audio File Library window pops up, click the Call Me to Record button.

4. Enter a name for your audio file. Make sure to include the language in the file's name for easy identification later.

5. Enter the direct dial phone number where you are located.

6. When the system calls, follow the menu prompts for recording your message.

7. When you are finished recording, your message will appear in the Audio Files list (see figure 8).

8. If you wish to edit the name of your recording, click the Edit link to the right of the file.

9. When you are finished, click the Done button in the Audio File Library.

 

Figure 8

Building Your Multilingual Phone Messages

10. Click the Create New Phone Message button (see figure 7).

11. Enter a brief name and description for your message in the message information section of the Phone Message Editor screen (see figure 9). Be sure to indicate the language in the message’s name for easy identification when you create your notification job.

12. Select your audio file from the Audio Recording drop down menu on the right side.

13. Click the Play to preview the audio file.

14. Click the Insert button to place the file in your message. Your audio file will appear in the Message Content area {{like so}}.

Important: Do not edit the information in the {{brackets}}. Doing so may cause your message to not work properly.

15. Click the Save button at the top or bottom of the screen.

16. Repeat these steps to assemble your message for another language. Proceed to the next section when you are done.

Figure 9

Schedule and Start Your Job

17. Click the Jobs tab to open the Notification Jobs page.

18. Click the Create New Job button in the My Active and Pending Jobs section.

19. Enter a name and description for your job in the settings section (see figure 10).

 

Figure 10

20. Select the Job Priority.

21. Select the list that you wish to contact.

22. Enter the start and end dates for the job. Most standard date formats are recognized (ex. 02/25/05 or February 25th, 2005).

23. Select the Send Phone Calls checkbox.

24. Select the Default Message that you wish to send. Note: Unless multilingual message options are specified this will be the message that is sent to all recipients.

25. Set the earliest and latest time that you want the system to attempt calls.

26. Set the maximum number of attempts that you want the system to make for busy and unanswered phone numbers.

27. If your system has multiple phone numbers in the database for each contact you can specify that it try every available number by selecting the option.

28. In the Multilingual Phone Message Options specify a language preference and message combination that you want to use in place of the default message you selected in step 24, and then click the Add button. For example, for Spanish you would select Spanish for the Language Preference, then your Spanish message for the Message to Send, and then click the Add button. Repeat this step for any additional language/message preferences that you wish to specify (see figure 11).

29. Click the Submit Job button to submit your job for processing.

 

Figure 11

From the Start tab or from the Jobs tab you can view the status of this job. You can also get an up-to-the-minute report of the job progress by clicking the Report link located to the right of the Job name.
