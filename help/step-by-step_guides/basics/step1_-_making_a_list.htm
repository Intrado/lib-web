Step 1: Making a List

The first thing you need to do before you can send any messages with the system is to make a list. This tutorial walks beginning users through the steps of creating a list. Most users will find that after creating a few simple lists they can create any number of complex lists without further assistance.

The best thing to do when learning how to use the system is to call yourself. This tutorial will guide you through the process of making a list that includes only you. For more advanced list creation, please read through the list creation help topics.

Remember: Lists can be reused! It's not necessary to create a new list for every job. You can use the list created in this tutorial for every schoolwide general announcement as well as for emergency calls. The list you create in this tutorial can be used to test any message you create in the future.

Create a New List

  1. Click the Notifications tab and then the Lists subtab.

  2. Click the Create New List button. This will take you to the List Editor screen, shown below.

  3. Enter a brief name and description of your list in the fields provided, then click Save. You will be taken to a screen where you can enter your information.

  1. Click the Enter Contacts button.

  2. Enter your contact information. Don't worry about filling in every field. For this tutorial, you will only need to enter your name, a phone number, and an email address.

  3. Click Save when you are done. This will take you back to the List Editor.

  4. Click Done. Now you're ready to proceed to Step 2 - Send an EasyCall.
