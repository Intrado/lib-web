Step-by-Step: Send a Phone and Email Job

You can send a job with a phone message and email message by following the instructions in this guide.

This tutorial walks beginning users through the steps of creating a list, creating both a phone and email message and then using those messages in a notification job launched from the system’s web interface.

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

 

Create a List

Note: If you have already created the list that you wish to use skip this section. If you want to use a list building method not included in this guide please refer to the Creating a List main help section.

Before you can send any type of notification you must create a list. This list will determine who receives the message you send. Lists can either be dynamic, static, or a combination of both.

  • Dynamic List - A list that is based on rules and that automatically updates as the data in the system’s database changes. For example, to create a list that only includes the 12th grade students you could simply use the rule 'Grade is 12'. This is the best way to call large groups, such as an entire school.

  • Static List - A list built either by selecting individuals from the system database or by manually typing individuals into the list. The people included on this list will not change unless you manually edit the list. For example, a list of PTA parents could be created by manually selecting each PTA parent that you wish to include on the list. This is the best way to add an individual, such as yourself, to the recipient list.

To Create a List

  1. Click the Notifications tab and then the Lists tab. This will open your List Builder page (See Figure 1).

  2. Click the Create New List button. This will open the List Editor.

  3. Enter a name and description for your list in the fields at the top of the List Editor screen.

Adding Rules to a List (creating a dynamic list)
Use this section to create a list using fields from your database that dynamically updates every time you use it.

  1. To create a dynamic list, start by selecting the field that you wish to base the list on. A list of available data items for that field will display when you click on the Select a Field drop down menu.

  2. Select the comparison operator to use, and then select the value(s) to include with this rule. You can select multiple items by holding the Ctrl key (command key on the Mac) while clicking. For example, 'Grade is 12' will create a list of all 12th graders.

  3. Click the Add button to add this rule to the list. Add additional rules to this list as required (See Figure 2).

  4. When you have added all of the rules you want to include, check the Total People in List value in the List Information at section at the top of the page to insure that it matches your expected result. Skip to the Previewing and Saving Your List if you are finished with this list.

Manually Adding People to Your List (creating a static list)
Use this section to add individuals to your list. This is a great way to include yourself in your list so that you are called along with everyone else when your job runs.

You can add a new person by clicking the Enter Contacts button in the Manual Add section of the List Editor screen.

 

To Manually Add Individuals To Your List:

  1. In the List Editor screen, click the Enter Contacts button in the Manual Add section. You will be taken to the Enter Contact Information screen shown above.

  2. To automatically add the individual's contact information to your Address Book, make sure the "Save to My Address Book" option at the bottom of the screen is checked.

  3. Click the Save & Add Another button in the upper or lower left corner to add the individual to your list and enter contact information for another person. If you don't need to add another person, proceed to step 4.

  4. When your list is complete, click the Save & Done button, located in the upper or lower left corners to save your list, or continue to add individuals using a different method.

Previewing and Saving Your List

  1. When you've finished adding people to your list, note the current total number of people in your list displayed near the top of the editor window. Click the Preview button to display the current list. You can remove any individuals whom you don’t want to contact by unchecking the box to the left of their name.

  2. Click Done when finished.

  3. Click Save to save this list and return to your List Builder page.

 

Creating a Phone Message with Call Me to Record

  1. Click the Call Me to Record button on the Message Builder page.

  2. Enter a name for your new message and the direct access phone number where you can be reached.

  3. Click Call Me to Record.

  4. You will receive a call from the system momentarily.

  5. Follow the prompts to record your message. To record more than one message, simply follow the prompts.

  6. When you are done, your new messages will appear in the Call Me window, as shown below, where you can assign them names.

  7. Click Save when you are done to save your messages and exit Call Me to Record.

 

Creating an Email Message

To create a new email message, click the Create New Email Message button on the Message & Survey Template Builder page as shown in the image below. This will take you to the Email Message Editor screen shown in the following image. If you plan to create this message in an alternate language be sure to indicate the language in the message name for easy identification when you create your notification job.

Writing Your Email:

  1. Enter a subject in the Subject field. This is what the recipient will see as the subject line of the email.

  2. Enter the name of the sender in the From Name field. This is who the recipient will see as the sender.

  3. Enter the sender's email address in the From Email field. This is the sender's email address as seen by the recipient.

  4. Enter the body of the email message in the Body field. You can include dynamic data fields at any point in your message. You may also use carriage returns for formatting.

  5. Click the Save button at the top or bottom of the screen when you are done. Your message will be saved and you will be returned to the Message Builder screen.

Scheduling Your Notification Job

To Create a New Notification Job:

In the My Active and Pending Jobs section at the top of the screen, click the Create New Job button which will take you to the Job Editor Screen, shown in the next image. You will see several fields divided into four major sections on the Job Editor screen. Please use the following explanation of the fields to help you enter the information for your job.

Settings:

  • Name: This field shows the name of the job you are about to send.

  • Description: Here you can add an optional description for your job.

  • Job Type: Select the type of job you are sending.

  • List: Use this drop down menu to choose which saved list you wish to send your message to.

  • Start Date: This field allows you to set the date your job will begin to run on. By default it contains the current date.

  • Delivery Window: This section establishes the times when the system will call your recipients.

  • Email a report when the job completes: If this box is checked, the system will email a PDF file with the results of the job to the email specified in your user account.

 

Phone:

  • Send Phone Calls: Check this box if you wish to send phone calls as part of this notification. It will also automatically set if you begin to configure other settings in the Phone section.

  • Default Message: This is the message that will be sent by default.

  • Multilingual Message Options: Use these menus to send a multilingual message if you created messages for other languages. Select the language of your alternate message and the name of your alternate message from the drop down menus. Anyone in your database who prefers that language, will receive the message you select here.

  • Maximum attempts: Use this drop down menu to select how many times the system should try to call each recipient.

  • Caller ID: This is the number that recipients will see on their caller ID.

  • Skip Duplicate Phone Numbers: Check this box if you want each phone number in your list to be called only once. For example, if you were calling every student in your school with a general announcement, you might want to skip duplicate numbers. Then if there are multiple students from the same house, that house will only be called once rather than repeating for each student in the house.

  • Call every available phone number for each person: Checking this box will cause the system to attempt to reach your recipients at every available phone number in the system. For example, if the database contained a home, work, and cell phone number all of them would receive a call.

 

Email:

  • Send Emails: Check this box if you wish to send emails as part of this notification. It will also automatically set if you begin to configure other settings in the Email section.

  • Default Message: Use the drop down menu to select the message that will be sent by default.

  • Multilingual Message Options: Use these menus to send a multilingual message if you created messages for other languages. Select the language of your alternate message and the name of your alternate message from the drop down menus. Anyone in your database who prefers that language, will receive the message you select here.

  • Skip Duplicate Email Addresses: Check this box if you want the system to automatically skip duplicate email addresses in your list.

 

Review and Confirm:

When you have finished configuring your job and are ready to send it, click the Proceed to Confirmation button. You will see a screen like the one below. If everything looks correct, click the Submit Job button. If you need to make a change, just click the back button in your browser.
