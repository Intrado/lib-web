Sending an SMS Job

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

If enabled on your account, you can send a short text message to your recipients using an SMS Job. An SMS message is a text message which can be received on mobile phones.

To Send an SMS Job

  1. Create a list of recipients.

  2. Click the Notifications tab.

  3. Click the SMS subtab. You will be taken to the SMS Job Editor, as shown below.

  4. Enter a Name and Description for your SMS job.

  5. Select the list you wish to contact.

  6. Enter the message. SMS messages cannot be more than 160 characters long.

  7. Double check all of your settings and click Submit Job.
