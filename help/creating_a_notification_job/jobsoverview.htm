Job Overview

Before attempting to start a new notification you must create a list of the people that you wish to contact and you must create the message that you want to send. If you have not done these two things begin by referencing the help sections on creating a list and creating a message prior to creating your notification. If you plan to call in to the system to create your job, you do not have to record a message before you begin.

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.
