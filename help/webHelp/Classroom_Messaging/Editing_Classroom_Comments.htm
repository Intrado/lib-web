Editing Classroom Comments

Important: This feature is only visible if your system includes the Classroom Messaging feature. For more information regarding the Classroom Messaging feature, please call 1-800-920-3897.

All of the 201 default classroom comment messages contains a professionally recorded audio version of the message for use with phone messaging. These are available for each language we're currently able to support, shown below. The ability to edit classroom comments is intended  to allow you to easily add comments in languages which are currently not supported.

Editing a Classroom Comment

  1. From the Classroom Message Manager, find the comment you wish to edit.

  2. Click the edit link next to the comment.

  3. Find the language you wish to add in the list of languages and click the Edit button next to the language. This will take you to the Classroom Message Editor.

  4. Enter the text version of the message which will be used in email.

  5. Create a phone message using the simple phone message editor in the lower half of the screen. The editor behaves the same as it does when you create a phone message.

Important: If the language does not have a corresponding text-to-speech voice, the English text-to-speech voice will be used by default. In this situation, for best results, you should use the "Call Me to Record" or Audio Upload feature and use a recording of an actual speaker.
