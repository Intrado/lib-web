Creating Custom Comments

Important: This feature is only visible if your system includes the Classroom Messaging feature. For more information regarding the Classroom Messaging feature, please call 1-800-920-3897.

You can also create your own custom comments from the Classroom Message Manager.

Access the Classroom Message Manager:

  1. Click on the Admin tab followed by the Settings subtab.

  2. Click the Classroom Message Manager link in the Features section.

 

Creating the New Comment

Note: Keep in mind the new comment  will be used within the Email and Phone  templates. For best results, format it like the other Classroom Comments. For example, "Was very helpful today."

  1. Select the category your message should be in.

  2. Click the Create Message button.

  3. Click the Edit button next to the default language (typically English) at the top of the list to enter the Classroom Message Editor, shown below.

  4. In the Classroom Message Editor:

    1. Enter the email message in the Email field at the top.

    2. In the Phone section, you can enter text  to be read by a text-to-speech voice, use Call Me to Record to add an audio message, or upload a custom audio file. This part of the editor behaves like the phone message content advanced editor.

      • If you just type your message in the Message field, it will be read by a text-to-speech voice.

      • Call Me to Record lets you record audio via the phone.

      • You can also use advanced features such as data field insertion and mixing audio files with data. To learn more about how data fields and advanced phone messages work, please read the help topic on creating advanced phone message content.

    3. Once you've created the audio part of your message, you can click the Preview button to listen to how it sounds.

    4. Click Save when you are done.

  5. Repeat this process for any additional languages.
