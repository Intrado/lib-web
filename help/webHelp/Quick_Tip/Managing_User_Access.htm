Quick Tip Administration

Important: Only those features that you have been granted access to will display. For access to additional features contact your system administrator.

User Management

Access Profile Options

Once the Quick Tip feature is enabled on your system, a new section called Quick Tip Controls with two options.

  • Topic Recipient: Topic Recipients are able to view tips which have been submitted to the system. Users with this option enabled will receive an email each time a tip is submitted.

  • Manage Topics: This option allows users to access the Topic Manager.

Strategies for Managing Access

Because users who are topic recipients receive an email each time a tip is submitted, you may find that it's best to create separate custom access profiles for these users. For example, by default, if a user is a topic recipient, but their account is not limited to data from particular schools, she will receive an email each time a tip is submitted to any school in your district. If you would prefer to designate certain users as tip recipients for particular schools or organizations instead, you should create a separate access profile for each of these users.

Organization Names

The imported names of your organizations are used in the drop down menu that tip submitters use when selecting which organization their tip is about. It is therefore important that your organization names are user friendly. For example, if you are a school district whose schools are labeled by number, tip submitters will have a difficult time selecting the correct organization. If your organizations need friendly names for Quick Tip, please contact support for assistance.
