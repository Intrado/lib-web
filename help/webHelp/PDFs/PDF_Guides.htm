PDF Guides

We've created several printable PDF guides to help you learn how to use SchoolMessenger or teach others.

Important: Some of these guides contain instructions for features you may not have access to or which may not currently be part of your account. For access to additional features, please contact your system administrator.  

Introduction

  • Quick Start for New Users: A simple, introductory walk-through guide where new users learn to configure their user account and then set up and send a test voice notification.

User Training Guides

  • Getting Started: This PDF has everything you need to know for sending a message, including list building and a walk through MessageSender.

  • Advanced Training: This PDF guide contains some of the more advanced features you can use to enhance your notification.

  • Creating a Survey: This PDF worksheet and guide helps you plan and prepare a survey.

System Administrator Training Guide

  • System Administrator Training: This PDF guide is intended for system administrators and covers topics such as user management, system configuration, and data imports.

Pocket Guides

  • Remote Access Pocket Guide: This PDF contains pocket sized references for logging in and using the system by phone.
