User Name and Password: Used when accessing the system via a web browser. Passwords are case sensitive.  Usernames and Passwords cannot be the same as your first and last name or each other.  They cannot be based on a dictionary word.

Telephone User ID # and Telephone PIN Code #: Used to authenticate users who call into the system. Only numeric values are allowed and must be 4 digits in length.  User ID and Pin code cannot be the same and cannot be sequential numbers. EX. 12345.
